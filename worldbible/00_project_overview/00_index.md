# 00 项目总纲索引

## 模块目标

用于维护项目级方向、设计原则、范围边界、当前状态与范围地图。

本模块负责项目级方向统一，不负责具体剧情与细节设定。

## 当前状态

- 状态：已建立项目总纲、当前状态页与范围校准文件
- 内容深度：项目级规范层
- 是否允许扩写具体设定：不允许；本模块只负责项目级护栏与导航

## 本模块范围

- 明确仓库总目标与总边界
- 明确当前重点对象与整体仓库范围的关系
- 明确四层范围划分：宇宙通则层 / 九主世界与主世界层 / 故事主世界层 / 游戏聚焦层
- 维护当前状态、下一步动作与跨模块协作入口

## 文件索引

- `01_project_brief.md`：项目级定位、目标、工作方式与总纲摘要。
- `02_design_principles.md`：设计原则、判断标准与长期一致性要求。
- `03_scope_and_constraints.md`：当前阶段允许内容、禁止内容与项目边界约束。
- `04_current_state.md`：当前有效状态、当前推进重点、下一步动作与锁定原则。
- `05_scope_map.md`：仓库范围地图、目录层级归属与常见跑偏风险说明。
