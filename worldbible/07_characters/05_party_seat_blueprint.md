# 05 主角团七席位角色原型

## 1. 文档用途

本文档用于为主角团建立七个席位的第一轮原型卡。

当前阶段只回答“这七个位子分别承接什么世界接口、来自什么来源模型、为什么必须存在”，不写正式姓名、不写完整生平、不写剧情桥段。

## 2. 使用边界

- 这不是人物传记文件。
- 这不是战斗职业槽位表。
- 这里的重点是“世界接口分工”与“角色原型差异”。
- 后续若要写具体角色，应先保住本文件里的席位边界，再下沉到人物层。

## 3. 七个席位原型卡

### 席位一：主角位

1. 席位名称
主角位

2. 主要来源模型
高含金量飞升者来源。

3. 主要世界接口
承接宇宙广度、飞升制度、小世界模板差异与“从低位世界触规后进入主世界”的核心体验。

4. 主要社会接口
以外来者身份切入古裂残天的中心秩序、地方结构与社会不适应性，是观察主世界现实的第一视角。

5. 主要历史接口
通过“进入一个高位但带伤未愈的主世界”，感知古裂残天的修复期、名义中心与历史裂伤余波。

6. 为什么此席位不可缺
没有主角位，玩家就失去贯穿整个宇宙与故事主世界的主视角，飞升制度也失去最核心的人物承载体。

7. 容易与哪个席位重叠
最容易与飞升同伴位 A、飞升同伴位 B 重叠。

8. 如何避免写重
主角位必须承担“玩家主视角 + 第一宇宙接口”功能，不能只是三个飞升者里其中一个更重要的人。

### 席位二：飞升同伴位 A

1. 席位名称
飞升同伴位 A

2. 主要来源模型
飞升者来源，但必须来自与主角位明显不同的小世界主模板方向。

3. 主要世界接口
承接另一类小世界道路源流与法的组织方式，证明“飞升者”并不只有一种生成路径。

4. 主要社会接口
通过外来者的第二视角，对比主角位在古裂残天的适应路径与社会切入点差异。

5. 主要历史接口
承接九主世界差异或小世界模板差异带来的“不同文明训练背景”与“不同上桌资格”。

6. 为什么此席位不可缺
没有这个席位，主角团会把飞升者经验误写成单一路径，宇宙广度会明显缩水。

7. 容易与哪个席位重叠
最容易与主角位、飞升同伴位 B 重叠。

8. 如何避免写重
必须在主模板、副模板、法的组织方式和社会来源上与另外两个飞升位拉开，而不是只换性格。

### 席位三：飞升同伴位 B

1. 席位名称
飞升同伴位 B

2. 主要来源模型
第三类飞升者来源，继续拉开小世界模板、法的组织方式与社会来源差异。

3. 主要世界接口
承接更偏边缘、高风险、异质或非主流的小世界成长路径，使飞升者内部形成真正的三角结构。

4. 主要社会接口
进入古裂残天后，可与主角位、飞升同伴位 A 在中心接受度、地方适配度或高风险区适应方式上形成强差异。

5. 主要历史接口
承接“不是所有高含金量飞升者都出自同一种文明训练逻辑”的历史与道路证据。

6. 为什么此席位不可缺
没有这个席位，前三个飞升位很容易只剩“不同强者”而不是“三种不同宇宙入口”。

7. 容易与哪个席位重叠
最容易与主角位、飞升同伴位 A 重叠，也可能和灰线历史位在“边缘感”上重叠。

8. 如何避免写重
强调其来源模型更偏异质道路、边缘模板或非正统成长逻辑，而不是再写一个主流飞升天才。

### 席位四：中心体系位

1. 席位名称
中心体系位

2. 主要来源模型
本土修士来源中的中心法统 / 核心区制度人物来源。

3. 主要世界接口
承接隔绝地域内部仍然延续下来的中心秩序残余、核心区外缘控制链条、守法宝一系接口与名义中心投影。

4. 主要社会接口
承接隔绝地域内部的高组织度残秩序、中心执行链条与守护残余，使主角团真正进入此地中心接口，而不是只从外围旁观。

5. 主要历史接口
承接残胜后的名义中心如何在隔绝地域内留下守序残余、合法性余绪与执行力衰减后的现实矛盾。

6. 为什么此席位不可缺
没有中心体系位，主角团会失去进入中心结构、理解高位主导格局与感知中心边界的关键窗口。

7. 容易与哪个席位重叠
最容易与地方接口位、灰线历史位重叠。

8. 如何避免写重
不能把其写成中心绝对代言人，也不能写成单纯反派代言；重点是“中心接口”，不是“中心立场”。

### 席位五：地方接口位

1. 席位名称
地方接口位

2. 主要来源模型
地方势力来源 / 本土修士来源中的地方区人物。

3. 主要世界接口
承接隔绝地域内部的地方区、凡俗聚落、小势力、小宗门、资源区与现实承接层。

4. 主要社会接口
让隔绝地域内部“依附坐大为主、区域自立为辅”的地方格局真正上桌，而不是只停留在主世界总层说明里。

5. 主要历史接口
承接大战后这片地域里的旧制残枝如何落地成现实组织、地方秩序为何无法被完全收回、以及谁在托底生存。

6. 为什么此席位不可缺
没有地方接口位，古裂残天会被误写成只剩中心和暗线的空心主世界。

7. 容易与哪个席位重叠
最容易与中心体系位、边缘流动位重叠。

8. 如何避免写重
地方接口位必须立足地方区的日常运转、凡俗国家与地方共同体，而不是退化成“弱化版中心人物”或“流动型人物”。

### 席位六：边缘流动位

1. 席位名称
边缘流动位

2. 主要来源模型
本土修士 / 地方势力来源中的群陆外海、商路、资源区、节点与空间入口带人物。

3. 主要世界接口
承接隔绝地域内部残存通路、群陆外海、商路、资源区、危险节点与空间入口带，使当前舞台不被误写成纯内陆中心区。

4. 主要社会接口
承接流动网络、节点秩序、资源区运转与入口带周边的高风险秩序市场现实结构。

5. 主要历史接口
承接大战后隔绝地域边缘与入口带如何成为地方势力、灰色关系与暗线渗透长期盘踞之地。

6. 为什么此席位不可缺
没有边缘流动位，主角团将难以接入古裂残天最动态、最现实、最不中心化的一层世界。

7. 容易与哪个席位重叠
最容易与地方接口位、灰线历史位重叠。

8. 如何避免写重
边缘流动位必须强调“流动网络 / 节点 / 资源 / 入口”，而不是泛化成另一个地方人物或地下人物。

### 席位七：灰线历史位

1. 席位名称
灰线历史位

2. 主要来源模型
反抗网络来源 + 历史承载型来源的合并席位。

3. 主要世界接口
承接隔绝地域内部的反抗网络、旧势力余波、历史断层、地下结构与大战残响。

4. 主要社会接口
承接反抗网络如何在此地渗透利用地方共同体、商路、资源区与入口带，又如何在局部与地方结构长期共生。

5. 主要历史接口
承接双极旧秩序余波在此地留下的长期裂伤，以及旧敌残余与暗线力量如何从分散残余逐步汇流为长期渗透网络。

6. 为什么此席位不可缺
没有灰线历史位，古裂残天的地下结构、反抗网络与历史张力就只能留在背景板上，无法进入主角团内部视野。

7. 容易与哪个席位重叠
最容易与边缘流动位、中心体系位重叠，也可能与飞升同伴位 B 在“边缘感”上重叠。

8. 如何避免写重
不能把它写成纯设定说明器，也不能只剩地下标签；必须同时承接“灰线结构”与“历史余波”，并保有行动性。

## 4. 当前约束

- 七个席位各自承担不同世界接口，不能只靠性格区分。
- 七个席位不是战斗职业槽位，后续具体人物设计也不得把它们硬化成职业分工表。
- 若后续具体人物设计无法对应七个席位中的至少一个核心接口，应优先回到本文件调整原型而不是硬写人物。
