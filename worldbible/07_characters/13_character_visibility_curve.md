# 13 角色可见度曲线

## 1. 文档用途

本文档用于整理七个角色概念在前 / 中 / 后段的显性程度变化。

当前阶段只写结构，不写具体剧情桥段。

## 2. 前段显性较高的角色

- 守序飞升者
- 中心执行者
- 地方承接者

原因：

- 这三者最能尽快建立“飞升后进入高位主世界”的基本现实。

## 3. 中段逐步抬高的角色

- 剑器孤行者
- 流动节点者

原因：

- 这两类角色更适合在主世界现实已初步建立后，再把道路差异与流动网络社会扩展开。

## 4. 先作为接口出现，后转为真正核心同伴的角色

- 愿力飞升者
- 裂史携行者

原因：

- 这两类角色更适合先以异质秩序接口与灰线历史接口出现，再逐步进入主角团核心层。

## 5. 当前约束

- 可见度曲线不等于剧情加入顺序定稿。
- 若后续故事结构变化，应优先保留“世界接口逐步展开”的原则。
