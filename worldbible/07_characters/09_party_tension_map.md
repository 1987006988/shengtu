# 09 主角团结构张力图

## 1. 文档用途

本文档用于整理主角团七张角色原型卡之间的结构张力，确保七人不是“同一类人换皮”，并为后续具体角色设计提供关系边界。

## 2. 哪几组最容易互补

### 宗门飞升者 + 中心执行者

- 前者提供外来者第一观察口，后者提供中心内部接口。
- 这组最适合承接“高位秩序到底如何现实运转”。

### 地方承接者 + 流动节点者

- 前者代表地方治理社会，后者代表流动网络社会。
- 这组最适合共同打开古裂残天中心之外的现实结构。

### 神道飞升者 + 裂史携行者

- 前者承接异质飞升与信念型秩序逻辑，后者承接大战余波与灰线网络。
- 这组最适合打开“正统叙事之外还有哪些历史与社会接口”。

### 剑器飞升者 + 流动节点者

- 前者强调道路纯度与单线深化，后者强调现实流动与节点网络。
- 这组最适合形成“高纯度个人道路”与“高复杂度世界现实”之间的互补。

## 3. 哪几组最容易冲突

### 中心执行者 + 裂史携行者

- 张力来源：一个承接名义中心，一个承接胜者叙事之外的历史余波。
- 这组冲突主要来自历史解释权、合法性判断与对当前秩序的容忍边界。

### 宗门飞升者 + 剑器飞升者

- 张力来源：同为飞升者，但道路组织方式、文明训练逻辑与进入主世界后的适配路径不同。
- 这组冲突主要来自“什么才算更完整、更纯、更有效的道路”。

### 中心执行者 + 地方承接者

- 张力来源：一个更接近中心秩序的执行逻辑，一个更接近地方现实的承接逻辑。
- 这组冲突主要来自“秩序如何落实”而不是“秩序是否存在”。

### 流动节点者 + 裂史携行者

- 张力来源：二者都接近灰色地带，但一个立足流动秩序，一个立足历史灰线。
- 这组冲突主要来自“灰色关系是生存结构还是结构裂伤的后遗物”。

## 4. 哪几组最容易写重

### 三个飞升位

- 风险：都被写成高含金量飞升者，只剩来源标签不同。
- 解决方式：必须拉开主模板、法的组织方式、社会来源与进入主世界后的适配路径。

### 中心执行者 / 地方承接者

- 风险：都被写成“本土向导”。
- 解决方式：前者负责中心执行链条，后者负责地方承接现实，问题意识必须不同。

### 流动节点者 / 裂史携行者

- 风险：都被写成边缘人物、灰色人物。
- 解决方式：前者强调流动秩序与节点社会，后者强调历史余波与地下灰线。

## 5. 如何保证七人都不是“同一类人换皮”

- 先区分他们接入的是哪一层世界结构，而不是先区分性格。
- 先区分他们承接的是哪一种社会位置，而不是先区分说话方式。
- 先区分他们面对的是哪一种历史问题，而不是先区分态度标签。
- 保持“飞升者三角 + 中心 / 地方 / 边缘 / 灰线四角”的结构，不允许任一角塌陷。

## 6. 哪些张力来自宇宙差异

- 三个飞升位之间的差异主要来自小世界模板差异、道路源流差异与飞升路径差异。
- 宗门飞升者、剑器飞升者、神道飞升者之间的张力，首先是宇宙广度带来的差异。

## 7. 哪些张力来自古裂残天现实格局

- 中心执行者、地方承接者、流动节点者之间的差异主要来自古裂残天的大势力主导格局、四级社会模型与空间结构。
- 他们的张力首先是“同一主世界不同区段如何运转”的现实张力。

## 8. 哪些张力来自历史余波

- 裂史携行者与中心执行者的冲突，首先来自大战后的合法性断层与历史叙述差异。
- 神道飞升者与裂史携行者的互补，也部分来自历史余波与灰线结构的共同接口。

## 9. 当前约束

- 本文件只整理结构张力，不进入具体剧情桥段。
- 后续若具体角色设计削弱了这里的张力来源，应优先回退到原型卡层重新检查。
