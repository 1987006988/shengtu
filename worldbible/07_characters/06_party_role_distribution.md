# 06 主角团席位分布说明

## 1. 文档用途

本文档用于说明主角团为何采用 `1 主角位 + 6 关键同伴位` 的整体结构，以及为何采用 `3 飞升者 / 3 本土灰线人物` 的来源比例。

## 2. 为什么采用 1+6 结构

`1 主角位 + 6 关键同伴位` 的结构，当前最适合本项目，主要因为：

- 一个位子专门承接玩家主视角与第一宇宙接口。
- 六个位子足够覆盖古裂残天的中心层、地方层、边缘流动层与灰线层。
- 七席位能形成“少而完整”的世界接口网络，既不太少导致世界层缺口过大，也不太多导致结构松散。
- 这一数量适合在不写成大群像拼盘的前提下，同时保住宇宙广度与故事聚焦。

## 3. 为什么采用 3 飞升者 / 3 本土灰线人物 的整体比例

当前正式比例可以理解为：

- 主角位 + 2 个飞升同伴位 = 3 个飞升者位。
- 中心体系位 + 地方接口位 + 边缘流动位 + 灰线历史位中，有 3 个明确属于本土 / 灰线人物层，整体形成约 `3 : 3` 的来源平衡。

这个比例合理的原因是：

- 飞升者负责宇宙广度、小世界模板差异与外来高含金量。
- 本土 / 灰线人物负责古裂残天现实结构、历史裂缝与社会落地。
- 若飞升者过多，主角团会变成“外来者观光团”。
- 若本土人物过多，宇宙广度与小世界模板差异会明显缩水。

## 4. 为什么必须有中心体系位

- 古裂残天是大势力主导型格局，且名义中心仍强。
- 没有中心体系位，主角团就很难真正理解中心秩序、核心区法统与执行力衰减的现实含义。
- 中心体系位同时也是古裂残天“仍然是高位主世界”这一点的重要人物承载体。
- 但这个席位只能保留 `1` 位，否则容易让主角团过度偏向中心叙事。

## 5. 为什么必须有地方接口位

- 地方区是古裂残天现实社会真正最厚的一层。
- 地方凡俗国家、地方共同体、地方强宗与资源区运转，是主世界现实感的重要来源。
- 没有地方接口位，古裂残天会被写得只剩中心与地下网络，中间层现实会塌掉。

## 6. 为什么必须有边缘流动位

- 古裂残天不是纯内陆中心世界，而是受损主大陆、裂散群陆、外海体系与大量附属空间层构成的复合主世界。
- 商路、资源区、节点与空间入口带属于制度层节点，不能只存在于背景说明中。
- 边缘流动位负责让群陆外海、资源流、节点秩序与高风险入口真正上桌。

## 7. 为什么灰线历史位要兼承反抗网络与历史余波

- 如果把反抗网络人物与历史承载型人物拆成两个独立席位，主角团会过于拥挤，也容易出现纯解释型角色。
- 古裂残天的灰线层，本来就是“旧势力余波 + 反抗网络渗透 + 历史断层残留”的重叠区。
- 因此，把灰线位设计成“灰线历史位”更合理：它既能打开地下结构，也能承接大战后的长期余波。

## 8. 哪些小世界主模板最适合优先放进前三个飞升位

当前更适合优先覆盖的方向是：

- 一个偏体系型模板：如正统宗门修仙模板或家族 / 势力经营模板方向。
- 一个偏单线深化型模板：如剑道 / 本命器 / 器修模板或斗战 / 体魄升阶模板方向。
- 一个偏异质 / 边缘型模板：如游历 / 志怪 / 公路仙侠模板、王朝 / 江湖 / 探案模板、诡异 / 污染模板方向。

这样能让前三个飞升位在世界观接口上明显拉开，而不是都变成“不同版本的主流飞升天才”。

## 9. 哪些来源不适合进入主角团，而更适合作为阵营人物或背景大人物

以下来源当前更适合留在阵营人物或背景大人物层，而不是直接塞进主角团：

- 过于纯粹的中心高位代言人。
- 只承担单一设定解释功能的旧势力残脉人物。
- 只负责展示世界上层压力、但缺乏长期同行理由的高层背景人物。
- 过于完整地代表某一单一阵营立场的人物。

主角团需要的是接口复合度，而不是职位高度本身。

## 10. 本文件对后续具体角色设计的约束

- 后续设计具体主角团成员时，必须先对应七席位中的某一个核心接口。
- 不得让三个飞升位都落到同一类小世界模板。
- 不得让本土位都沦为“向导”或“设定讲解员”。
- 不得把灰线历史位写成纯 exposition 工具人。
