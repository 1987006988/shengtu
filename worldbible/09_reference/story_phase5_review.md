# 第五阶段回顾

## 1. 阶段目标回看

本阶段目标是把第一卷从卷级骨架推进到可直接开写正文的大纲层。

## 2. 发现的问题

### 问题一：章纲容易只剩结构，不见人物动作

- 风险来源：已有大量骨架文件，继续拆分容易变成空转提纲。

### 问题二：场景卡容易滑入任务拆步

- 风险来源：章节和节点已经足够细，如果不强调“小说场景”，就会像流程表。

### 问题三：中段角色容易失重

- 风险来源：前段守序飞升者和地方承接者天然更显眼，后段灰线历史位和愿力飞升者更强，中段容易塌腰。

## 3. 已做修补

- 在章纲表中给每章同时绑定“功能、冲突、揭示、抬高角色”四列。
- 在场景卡中把类型限定为结构、冲突、揭示、转折四种小说场景，而非任务步骤。
- 通过第 6、9、11、14 章与对应场景，专门给剑器飞升者、流动节点者、愿力飞升者补权重。

## 4. 是否足够支撑正文

足够。

第一卷已经具备稳定的章节顺序、场景功能、信息阶梯和角色抬升线，可以直接转写正文初稿。
