## 3. 当前可信事实

### 3.1 项目和范围事实

- 项目中文总名为 `升途`。
- 英文正式名为 `Ascendant Path`。
- 第一款游戏 / 第一部标题为 `升途：封绝之地`。
- 第一款游戏主舞台为 `白鹿原`。
- `白鹿原` 是 `古裂残天` 内部隔绝地域，不等于整个 `古裂残天`。
- 当前工程目标是大型 CRPG 叙事文本总工程，不是小说短稿或单卷故事稿。
- 总蓝图最低文本目标为 `1,500,000` 汉字，目标区间为 `3,000,000–6,000,000` 汉字。

### 3.2 白鹿之灵事实

- 白鹿之灵是接引法宝之灵的白鹿显化。
- 它是主线核心装置的灵性接口。
- 它可以承担接口、残忆、触发器、示警和边界反馈功能。
- 它可以参与复生机制，但必须保留代价、边界与状态反馈。
- 它不得成为普通灵兽、独立神祇、世界意志、万能解释器、万能支援、万能真相机或无代价复生源。
