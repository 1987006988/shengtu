# 第一部与宇宙总范围说明

## 1. 文档用途

本文档用于区分：哪些内容属于宇宙长期正典，哪些内容属于 `古裂残天` 长期正典，哪些内容属于第一部 / 当前故事强绑定设定。

## 2. 宇宙长期正典

- 三层宇宙结构。
- `3000` 小世界与 `9` 主世界框架。
- 主世界之间严重隔离、不设稳定宇宙中心。
- `气 / 规则 / 法 / 锚点` 的底层力量逻辑。
- 小世界八主模板与“主模板 + 副模板”结构。
- 九主世界差异、历史路径差异与主世界层比较框架。

## 3. 古裂残天长期正典

- `古裂残天` 是古老残缺修复型主世界。
- 双极顶层势力共同主宰旧秩序，并在高烈度大战后一方残胜。
- 大势力主导型格局、名义中心仍强但执行力分层衰减。
- 四级社会模型：核心区 / 地方区 / 群陆外海区 / 空间入口带。

## 4. 第一部 / 当前故事强绑定设定

- 当前主线舞台是 `古裂残天` 内部一片被大战推离的隔绝地域。
- 主角团只是刚触摸规则便飞升的高含金量新人。
- 主角团当前只能对这片地域命运起决定性作用，不能直接决定整个主世界命运。
- 接引法宝是当前主线的核心装置，同时承担接引体系核心、复生根源与地域存续支点。
- 当前故事主轴是这片地域是否重连、重启、继续封闭或被重组，以及由谁承担代价。

## 5. 以后可以变化的部分

- 隔绝地域的正式命名。
- 接引法宝最终路径的收束方式。
- 第一部故事中的具体任务层、章节层、节点层与角色具体传记层。

## 6. 使用约束

- 若内容只属于第一部强绑定设定，不得回写成全宇宙通则。
- 若内容属于 `古裂残天` 长期正典，不得自动外推到其他主世界。
- 若内容属于宇宙长期正典，应优先落在 `01–06` 与 `09_reference` 的对应文件中。
