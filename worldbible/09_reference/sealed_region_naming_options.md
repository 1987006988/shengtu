# 白鹿原命名拍板记录

## 1. 文档用途

本文档用于记录《升途：封绝之地》当前主舞台对应的隔绝地域正式命名拍板结果、命名层级与既有候选复核记录。

当前有效口径：

- 第一部主舞台正式设定名：`白鹿原`。
- `白鹿原` 是第一部主舞台在设定正文中的正式地域名。
- “隔绝地域 / 封绝地域”是功能性描述，不替代正式地名。
- `离序残域` 不作为当前这片隔绝地域的正式地名写入正典。
- 如需保留，`离序残域` 仅可作为工作代号 / 档案代号 / 描述性标签使用。

命名层级区分如下：

- `升途`：项目总名。
- `升途：封绝之地`：第一部标题 / 叙事层称呼。
- `白鹿原`：第一部主舞台在设定正文中的正式地域名。

## 2. 命名前提约束

- 不能比项目总名 `升途` 更像系列总标题。
- 不能写得像整个 `古裂残天` 的主世界总名。
- 不能小到像单纯地图角落或普通任务区域。
- 要符合第一部舞台的高位残存、封闭悬置、裂伤未愈、仍被托底的气质。
- 命名必须遵守 `naming_rules.md` 与 `naming_review_checklist.md`。

## 3. 正式拍板名

### 白鹿原

- 拍板状态：已确认。
- 命名层级：第一部主舞台正式地域名。
- 使用边界：仅指 `古裂残天` 内部这片被大战推离主世界常规秩序带的隔绝地域，不等于整个 `古裂残天`，也不等于宇宙总层结构。
- 与标题关系：`升途：封绝之地` 保留为第一部标题 / 叙事层称呼；`白鹿原` 承担设定正文中的地域名功能。
- 与功能描述关系：“隔绝地域 / 封绝地域”可继续说明其结构状态，但不替代 `白鹿原`。

## 4. 回撤记录

### 离序残域

- 当前状态：不采用为正式地名。
- 可保留范围：工作代号 / 档案代号 / 描述性标签。
- 回撤原因：这片地域本质上仍是 `古裂残天` 的一部分，`离序残域` 的表达更像外部研究、守档或技术记录中的描述标签，不适合作为地域内部正式地名。
- 使用边界：不得在设定正文中写作“正式设定名”“正式地域名”或“已拍板命名”。

## 5. 候选复核记录

### 正式采用：白鹿原

- 气质说明：更像地域内部可被长期称呼的正式地名，同时与接引法宝之灵的白鹿显化形成内在呼应。
- 复核结论：已采用为正式设定名。

### 候选一：封绝遗域

- 气质说明：强调被封绝后的高位残存感，适合当前第一部舞台的整体气质。
- 复核结论：可用但未采用，偏稳妥，也因此略显保守。

### 候选二：离序残域

- 气质说明：突出它已经离开主世界常规秩序带，却仍保留残存结构。
- 复核结论：不推荐作为正式地名；若保留，仅适合作为工作代号 / 档案代号 / 描述性标签。

### 候选三：裂序遗境

- 气质说明：同时带出大战裂伤与残秩序留存，适合较强的历史伤痕感。
- 复核结论：可用但未采用，整体裂伤感偏重。

### 候选四：断引封域

- 气质说明：直接把接引断裂、封闭悬置与高位结构失联感压进名字里。
- 复核结论：未采用，和接引法宝关系过近。

### 候选五：悬绝遗域

- 气质说明：强调它处于主世界之外不外、之内不内的悬置状态。
- 复核结论：可用但未采用，信息略轻。

### 候选六：残带封域

- 气质说明：更强地指向“被推离主世界常规秩序带”的结构感。
- 复核结论：未采用，略近说明词。

## 6. 当前使用约束

- 正式正文中使用 `白鹿原` 指代第一部主舞台正式地域名。
- `隔绝地域 / 封绝地域` 继续作为功能性描述，用于说明其结构状态与故事尺度。
- `封绝之地` 保留为第一部标题 / 叙事层称呼，不自动转为地域正式名。
- 正式正文中不得把 `离序残域` 写作第一部主舞台正式地名。
