# 06 九主世界研究笔记

## 1. 文档用途

本文档用于记录本轮“九主世界统一初稿”与“故事主世界重点深描”所参考的公开资料类型、归纳方法与采用逻辑。

本文档不是正典正文。

- 外部作品与资料只用于提炼常见高质量做法。
- 本项目不会把任何单一作品的一整套结构直接搬运为正典。
- 正式采用结论以 `05_nine_main_worlds_draft.md` 与 `04_main_world/05_story_main_world_draft.md` 为准。

## 2. 调研样本来源与方法说明

本轮调研遵循以下方法：

- 不假装通读全部长篇全文。
- 优先查阅公开可访问的官方简介、百科、wiki、读者设定整理页。
- 样本选择以“大千世界 / 上界下界 / 多位面秩序 / 飞升上界”特征明显的高影响作品为主。
- 不抽取作品专有名词进入本项目正典，只抽取结构模式。
- 重点提炼“主世界统治形式、仙凡关系模型、空间结构、文明厚度和古老残缺型世界写法”。

本轮主要参考方向包括：

- 《大主宰》相关公开设定页：观察“大千世界 - 下位面”层级写法与上位位面的汇聚性。
- 《完美世界》相关公开设定页：观察“下界 / 上界 / 更高仙域”的空间承载差异与边界感。
- 《凡人修仙传》相关公开设定页：观察“人界 / 灵界 / 仙界”的分层飞升结构和界面差异。
- 《仙逆》相关公开设定页：观察多重高位区域与大尺度版图的组织方式。
- 《莽荒纪》相关公开设定页：观察“三千大世界、亿万小世界”式的大规模世界表达。
- 部分高热升级流世界整理页：补充天庭秩序、宗门共治、王朝法统、海陆网络与边界封控等主世界模型。

## 3. 主世界差异最适合落在哪些维度

从公开样本看，多个主世界若要真正成立，最适合拉开差异的不是单纯战力，而是以下六类维度：

1. 统治形式
是否由单一霸权、宗门共治、神庭秩序、王朝法统、技术寡头、边界中枢或混合议会维持。

2. 仙凡关系
是仙凡分离、深度混融、王朝代理统治、修士覆盖世俗，还是区域差异混合。

3. 世界结构
是单主大陆、主大陆 + 群陆、外海体系、并列多域，还是附属空间高度占比的复合结构。

4. 修行气质
是法统厚重、神道秩序、结构承载、尚武破限、边界封控、流动兼容，还是重遗产修复。

5. 社会气质
是庄严分层、强治理、开放流动、技术务实、边疆尚武、多族协商，还是古旧带伤。

6. 文明厚度来源
有的厚在制度，有的厚在法统，有的厚在空间层，有的厚在历史叠层，有的厚在残存遗产。

## 4. 主世界常见的统治形式

从样本归纳，主世界常见统治形式主要包括：

- 单一霸权：一个上层中枢长期统领全局，合法性来源明确。
- 宗门共治：多支顶层法统相互制衡，形成稳定上层秩序。
- 神庭 / 天庭秩序：以位格、册封、神职和秩序分层构成世界骨架。
- 王朝法统：王朝本身就是高位秩序的合法载体，修士可纳入体制。
- 技艺寡头：以工坊、学宫、炼制体系或传承机构把持文明中枢。
- 混合议会：多强权、多传统之间通过协商维持总体均衡。
- 封控中枢：围绕污染、异常、边界和稳定化治理形成秩序中心。
- 古制残存 + 新秩序拼接：旧日中枢尚在，但已无法完全恢复原有大一统统治。

本项目适合采用这类结构，是因为它们能够让九个主世界形成真正的“制度差异”，而不是只换皮。

## 5. 主世界里凡人与修士的常见相处模型

公开样本里最常见的几类模型如下：

### 5.1 仙凡分离

- 高位秩序在上层运转，凡俗王朝在下层承接治理。
- 适合天庭型、神道型主世界。

### 5.2 修士覆盖世俗

- 世俗仍存在，但核心安全、资源和秩序由修士掌控。
- 适合传统宗门主世界、边疆强压主世界。

### 5.3 王朝代理统治

- 凡俗国家机器不是背景板，而是高位秩序的实际代理层。
- 适合法朝型主世界。

### 5.4 深度混融

- 修士与凡人共同嵌入城市、商路、边疆、港城、官僚和家族结构。
- 适合海陆网络型、多域混合型、法朝型主世界。

### 5.5 区域差异混合

- 同一主世界内不同区域使用不同模型。
- 适合古老残缺修复型主世界，因为它通常无法再维持一刀切秩序。

## 6. 主世界的大陆 / 群陆 / 外海 / 附属空间常见结构

公开样本里，高位主世界常见的空间结构不是单一平面大陆，而更接近下列组合：

- 单主大陆 + 上下层秩序分层
- 主大陆 + 群陆 + 洞天群
- 群陆 + 外海 + 航道网络
- 多块并列大域 + 多传统附属空间
- 边界裂隙带 + 封存区 + 异常附属空间
- 破损主大陆 + 碎裂群陆 + 遗域密集附着

适合本项目的吸收方式是：

- 用空间结构直接服务于文明差异，而不是只服务于地图观感。
- 让附属空间层成为文明厚度的一部分，而不是随机副本。
- 让故事主世界的“裂伤未愈”表现为结构性空间问题，而不是单纯荒凉。

## 7. 高位主世界如何体现“文明厚度”

高位主世界的文明厚度，常见不只体现在“历史更长”，更体现在：

- 多时代叠层同时存在。
- 旧法统与新秩序彼此纠缠。
- 附属空间保存大量遗产与结构残留。
- 大规模基础设施可跨时代运转。
- 不同道路可以并置、互证、筛选和上推。
- 上层秩序能长期记录、归类和接纳飞升者。

因此本项目中“强大的主世界”不该只是更大、更富，而应显得更厚、更复杂、更能组织多种道路与历史沉积。

## 8. 古老残缺修复型主世界应如何处理

从外部样本归纳，如果“古老残缺型主世界”写不好，很容易滑向两个误区：

- 误区一：写成纯废土、纯末世，失去主世界应有的高位感。
- 误区二：只说曾经辉煌，却让当前世界没有任何实际结构后果。

更适合本项目的处理方式是：

- 它依然高位，依然能运转，依然有厚重遗产与完整文明残基。
- 它的问题是“旧日裂伤始终未被完全修复”，而不是“世界已经死了”。
- 大战留下的是结构性后果：空间裂伤、秩序断层、接引残缺、权力中枢能力下降、地方化抬头。
- 顶层势力不是消失，而是仍在，但已不能再像全盛时期那样恒压全域。
- 因此地方势力、旧势力、隐势力、新势力才会同时浮出，而这正是世界仍然活着、仍在再生的表现。

## 9. 哪些做法适合本项目

以下做法适合吸收：

- 用统治结构和仙凡关系模型拉开主世界差异。
- 让世界结构本身体现文明类型差异。
- 把主世界强度写成规则完整度、空间厚度、基础设施和文明厚度的综合强。
- 保留飞升者在主世界中的高含金量，不让他们到上界后只剩背景功能。
- 让故事主世界既高位又带伤，保持“修复中”的上升态。

## 10. 哪些做法不适合本项目

以下方向不适合采用：

- 把九个主世界分别写成九部小说的投影。
- 把差异只写成境界、数值和战力排名差异。
- 把古老残缺主世界写成废土末世或纯衰败世界。
- 把所有主世界写成同一套结构只换名字。
- 把灰色小世界写成随机副本，而不是秩序边界的一部分。

## 11. 本轮采用结论

本轮最终采用如下方向：

- 九个主世界全部统一为可比较的世界卡结构。
- 天阙序庭与皇极法朝明确保留不同的仙凡关系边界。
- 故事主世界正式收束到“古老残缺修复型主世界”。
- 故事主世界的核心表达，不是废土、不是末世、不是完全衰败，而是“仍然高位、仍然运转、但始终未能完全修复旧日裂伤的主世界”。
