# 气、规则、法调研笔记

## 1. 文档用途

本文件用于整理外部玄幻 / 仙侠作品中，与基础天地能量、高阶法则 / 本源 / 大道，以及修炼方法体系相关的公开设定做法，并为本项目正式文档的表达补强提供参考。

本文件不是正典，不直接改写本项目已确认结论。其用途是：

- 帮助补强 `02_power_system` 中“气 / 规则 / 法”的表述密度
- 为边界说明、常见误解澄清与非采纳方向提供外部参照
- 区分“外部作品启发”与“本项目正式口径”

## 2. 调研作品列表

本轮重点参考作品如下：

- 《盘龙》
- 《吞噬星空》
- 《雪鹰领主》
- 《仙逆》
- 《凡人修仙传》及《凡人修仙之仙界篇》
- 《莽荒纪》

补充说明：

- 主要参考公开百科、作品 wiki、公开设定页与读者整理页。
- 本轮重点关注“表达结构”和“概念组织方式”，不是逐字考据。

## 3. 各作品对“基础能量”的启发

### 《盘龙》

- 启发点：基础超凡力量与天地元素高度相关，修炼不是凭空生出力量，而是与天地中本已存在的要素发生联系。
- 可借鉴处：可把基础能量写成“天地常在、可被引动、可被分流、可被组织”的环境性存在。
- 不适合直接照搬处：元素分类和法则谱系太固定，容易把本项目“气本无定名”写成先天地分类表。

### 《吞噬星空》

- 启发点：基础力量与宇宙环境、生命结构、空间结构密切相关，能量并不是单纯燃料，而是与世界尺度、生命尺度共同运转。
- 可借鉴处：适合补强“气会随世界结构、环境演变与众生活动而流动变化”的世界学意味。
- 不适合直接照搬处：过强的宇宙工程感和体量设定，容易让“气”显得过于科幻化。

### 《雪鹰领主》

- 启发点：前期斗气、法力等具体形式，与后期真意、道之间并不割裂，说明基础能量与高阶秩序之间可以通过修行过程逐步打通。
- 可借鉴处：不同文明、不同世界对同一底层能量形成不同运用体系，是因为观察角度、修行传统和承载条件不同。
- 不适合直接照搬处：若把真意分品的逻辑移到“气”上，会让气被过度等级化。

### 《仙逆》

- 启发点：灵气、仙气等具体形态虽然不同，但都可以作为修行者进入更高层次的环境基础。
- 可借鉴处：适合强化“气可因环境、层级、世界条件不同而表现出显著偏性，但仍保有共同根源”。
- 不适合直接照搬处：若把不同层级环境的气完全割裂成彼此无关的几种本体，会冲击本项目已确认口径。

### 《凡人修仙传》

- 启发点：灵气、魔气、仙灵气等环境差异会显著塑造修行方式与文明结构。
- 可借鉴处：可以把“福地 / 绝地 / 隔绝区 / 稀薄区”的形成，与气的浓度、偏性、流动和环境条件联系起来。
- 不适合直接照搬处：不应把气写成某种完全依赖采掘、争夺、囤积的资源包。

### 《莽荒纪》

- 启发点：天地元力、混沌之力等常被视为世界常在的底层支撑，而不是一次性用品。
- 可借鉴处：适合补强“气恒常存于天地”这一点，让其更像世界运行的背景介质。
- 不适合直接照搬处：若过度强调某一种终极能量，会弱化本项目“气本无定名”的开放性。

## 4. 各作品对“高阶规则 / 法则 / 本源”的启发

### 《盘龙》

- 启发点：高阶修炼从能量积累明确转向法则玄奥理解，且法则是真实存在、可被逐步把握的客观结构。
- 可借鉴处：规则可以写成“被人认识得不统一，但其客观存在不因命名而改变”。
- 不适合直接照搬处：法则谱系和层次过于固定，容易让规则体系变成严格树状分级。

### 《吞噬星空》

- 启发点：后期修炼显著转向本源法则、宇宙奥秘与更高秩序结构。
- 可借鉴处：适合强化“规则不是抽象口号，而是可以被稳定接触、长期积累和逐步掌握的深层结构”。
- 不适合直接照搬处：不适合把规则写成无限展开、无限分叉、无限升级的庞大技术树。

### 《雪鹰领主》

- 启发点：真意、道、开辟道路等设定强调“从感悟到稳定道路”的过程。
- 可借鉴处：有助于补强“感知、映照、借用、掌控”四阶段关系。
- 不适合直接照搬处：不应把规则理解做成固定品级排名，否则会削弱“不同规则无绝对高下”的项目原则。

### 《仙逆》

- 启发点：后期从意境走向本源，说明高阶秩序应被写成真实存在、可被触摸但不等于可被完全命名的层面。
- 可借鉴处：适合加强“众生命名与规则本体并不等同”的表达。
- 不适合直接照搬处：不应让规则体系被极端主角化奇遇所主导。

### 《凡人修仙传》及《仙界篇》

- 启发点：后期明显转向法则与大道层面，且不同法则有不同显化与传承条件。
- 可借鉴处：很适合支持“规则数量有限，但可感知难度、显化频率、传承难度与表现方式不同”。
- 不适合直接照搬处：不适合把规则写成存在严格母子级上下统辖体系，否则会冲击“无绝对高下”。

### 《莽荒纪》

- 启发点：修炼最终走向“道”，而且不同道路可通向同等高层次，不必有绝对高下。
- 可借鉴处：可用来支撑“不同规则之间没有绝对高下，但稀有度与特殊性不同”的表达。
- 不适合直接照搬处：不能把“终极之道”式结构直接变成本项目唯一终点模板。

## 5. 各作品对“修炼方法体系 / 桥梁层”的启发

### 共通启发

- 无论作品使用“功法、神术、斗技、观想法、阵法、炼器法”哪种名称，这些内容通常都承担“把修行者从能量使用者推向高层秩序接触者”的桥梁作用。
- 真正决定高阶道路的，往往不是方法数量本身，而是方法能否帮助修行者持续接近某类高层秩序。

### 对本项目最有价值的启发

- 法可以很多，体系可以杂，但只要仍然承担“走近规则”的功能，就不会与本项目口径冲突。
- 法不必全都以战斗招式出现，也可以包括观想、祭炼、构筑、引导、炼体、养神、布置与组织气的方式。
- 高阶修炼中，法更像“桥”和“路径组合”，而不是最终本体。

### 不适合本项目的外部做法

- 把“法”重新写成一种与气、规则并列的独立原初力量。
- 把所有高阶方法都写成固定职业树或等级树。
- 把“法”写成只要获得传承文本就能直接跳过理解过程的现成答案。

## 6. 哪些写法适合本项目

- 把基础能量写成天地常在的环境性介质，而非一次性消耗包。
- 把高阶秩序写成真实存在、可被接触、可被误解命名但不因命名而改变的结构。
- 把修炼方法写成通往更高秩序的桥梁系统，而非第三本体。
- 强调世界差异、文明差异与修行传统差异，会让同一底层力量呈现不同名称与运用体系。
- 用条文化方式写清“定义、性质、边界、误解与非采纳方向”，有助于长期维护。

## 7. 哪些写法不适合本项目

- 把气写成一次性、可挖空、会彻底枯竭的外挂能源。
- 把规则写成无限分化、可以随意命名就算成立的标签集合。
- 把规则写成绝对等级森严的金字塔，本质上把某些规则固定为“天生高等”。
- 把法写成独立于气与规则之外的第三本体层。
- 把高阶修炼过度简化成境界到了就自动得到更高权限。

## 8. 对本项目“气 / 规则 / 法”表达补强的建议

### 对“气”的建议

- 增加“环境性、流动性、偏性、浓度、稀薄区、隔绝区、污染区、汇聚区”的条文化说明。
- 强调气不是单纯燃料，而是天地长期存在的背景介质。
- 增加“名称来自文明经验，不来自气的先天自报身份”的说明。

### 对“规则”的建议

- 增加“规则本体与众生命名不完全等同”的澄清。
- 增加“数量有限不等于人人都能枚举清楚”的边界说明。
- 增加“无绝对高下，但稀有度、诡异性、显化难度不同”的误解澄清。

### 对“法”的建议

- 增加“法既包括功法，也包括组织气、理解规则、构筑承载结构的方法体系”的说明。
- 增加“法很多、路不同，但最终都在接近规则”的结构表达。
- 在正式稿中补一节“法与锚点的区别”，避免后续混淆。
