# 02 力量体系索引

## 模块目标

用于维护宇宙通则层的底层力量逻辑，包括气、规则、法、修行者与规则关系、锚点等核心设定。

本模块服务整个宇宙，不应只为某一个主世界、某一个角色或某一条剧情线服务。

- 状态：第一版正式稿已完成
- 内容深度：宇宙通则层
