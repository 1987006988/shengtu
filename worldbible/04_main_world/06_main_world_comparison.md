# 06 九主世界横向比较

## 1. 文档用途

本文档用于横向比较九个主世界，帮助后续人工审阅哪些差异已经足够鲜明，哪些地方仍容易混淆。

## 2. 统治形式比较

- 天阙序庭：神庭 / 天庭单一霸权。
- 玄门上陆：宗门共治。
- 皇极法朝：王朝法统。
- 海天群陆：海陆强权议会。
- 万工灵域：技术型寡头共治。
- 战荒重陆：战盟霸权。
- 百族议界：多族混合议会。
- 幽垢边庭：封控型秩序中枢。
- 古裂残天：古制残存 + 新秩序拼接。

## 3. 仙凡关系比较

- 天阙序庭：仙凡分离最明显。
- 皇极法朝：修士与凡人深度混融，修士可入官僚体系。
- 玄门上陆：修真秩序强于世俗，但保留凡俗社会。
- 海天群陆：修士与凡俗长期嵌套在航道和港城网络中。
- 万工灵域：凡俗深度参与高阶文明运转。
- 战荒重陆：凡俗共同体长期嵌入边疆与守备结构。
- 百族议界：区域差异最大。
- 幽垢边庭：高压边地共处结构明显。
- 古裂残天：多模型并存，受修复进程影响最大。

## 4. 世界结构比较

- 天阙序庭：纵向分层结构。
- 玄门上陆：主大陆 + 群陆 + 洞天群。
- 皇极法朝：主大陆腹地 + 边陲大域。
- 海天群陆：群陆 + 外海 + 节点网络。
- 万工灵域：资源主核 + 工坊群陆 + 结构空间层。
- 战荒重陆：重陆腹地 + 边荒裂域。
- 百族议界：并列多域 + 群陆。
- 幽垢边庭：裂隙带 + 封存域 + 异常空间。
- 古裂残天：破损主大陆 + 碎裂群陆 + 高频附着遗域。

## 5. 修行气质比较

- 天阙序庭：秩序、册序、位格。
- 玄门上陆：法统、互证、总谱。
- 皇极法朝：法度、治理、功绩。
- 海天群陆：流动、历练、兼容。
- 万工灵域：结构、媒介、祭炼。
- 战荒重陆：高压、破限、体魄。
- 百族议界：杂糅、折中、跨传统比较。
- 幽垢边庭：稳定、代偿、风险识别。
- 古裂残天：遗产、修复、拾取、重建。

## 6. 社会气质比较

- 天阙序庭：庄严、威权、分层。
- 玄门上陆：复杂、竞争、阶序成熟。
- 皇极法朝：组织化、治理化、制度厚。
- 海天群陆：开放、交易导向、边界柔性。
- 万工灵域：务实、精密、分工强。
- 战荒重陆：尚武、粗粝、边疆感强。
- 百族议界：多元、复杂、协商成本高。
- 幽垢边庭：紧张、戒备、持续维稳。
- 古裂残天：古旧、层叠、带伤但不衰败。

## 7. 最容易混淆的组别

### 7.1 天阙序庭 vs 皇极法朝

- 相同点：都具强秩序、都不是纯散修格局。
- 核心差别：天阙序庭是上层神道秩序与下层凡俗王朝分层；皇极法朝是修士和凡人共同嵌入同一官僚体制。

### 7.2 玄门上陆 vs 万工灵域

- 相同点：都重传承、都不排斥体系化修行。
- 核心差别：玄门上陆重法统谱系和道路总谱；万工灵域重结构承载、技艺文明和组织制造能力。

### 7.3 百族议界 vs 海天群陆

- 相同点：都偏开放、都不是单一中心结构。
- 核心差别：百族议界的复杂性来自多族多传统并存；海天群陆的复杂性来自空间结构流动和航道网络。

### 7.4 幽垢边庭 vs 古裂残天

- 相同点：都与边界、异常、遗域相关。
- 核心差别：幽垢边庭是主动围绕边界封控形成秩序；古裂残天是大战后遗留裂伤导致的修复型复杂结构。

## 8. 差异最明显的组别

- 天阙序庭 vs 战荒重陆：一个秩序位格极强，一个高压斗战极强。
- 皇极法朝 vs 海天群陆：一个偏制度腹地，一个偏流动网络。
- 万工灵域 vs 幽垢边庭：一个偏结构制造，一个偏异常封控。
- 玄门上陆 vs 古裂残天：一个偏完整法统总谱，一个偏高位遗产修复。

## 9. 当前结论

- 九个主世界已经形成可比较骨架。
- 当前最需要人工继续审阅的，是天阙序庭 / 皇极法朝与古裂残天 / 幽垢边庭两组边界。
- 若这两组边界通过审阅，九主世界差异化框架就已经足够支撑后续故事主世界深描和区域扩写。
