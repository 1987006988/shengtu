# 13 白鹿原功能地图

## 1. 文档用途

本文档用于把白鹿原内部的故事承载关系压成“功能地图”，帮助后续主线、章纲和正文保持空间与冲突的一致性。

## 2. 白鹿原的功能分区

### 2.1 核心维持区

- 包含：法宝核心牵引面、残秩序缓冲圈
- 回答的问题：白鹿原为什么没有塌。
- 适合承载：白鹿之灵出现、法宝绑定、结构控制冲突、最终决断。

### 2.2 现实承接区

- 包含：主聚落承接面、资源与风险交换带
- 回答的问题：白鹿原今天为什么还活着。
- 适合承载：地方共同体、日常秩序、资源争夺、代价落地。

### 2.3 流动边界区

- 包含：入口残带、断续通路、节点经营面
- 回答的问题：白鹿原如何与自身各层保持最低限度联通。
- 适合承载：商路、流通、资格、渗透、追索和转运。

### 2.4 失稳潜伏区

- 包含：裂伤外缘、隐匿潜伏层
- 回答的问题：白鹿原未来会从哪里被改写。
- 适合承载：灰线、残忆、危险探索、真相翻面。

## 3. 主线移动逻辑

第一卷的空间推进应遵守：

1. 先从现实承接区立脚。
2. 再通过流动边界区看见白鹿原不是封死的静态箱体。
3. 之后被迫接触失稳潜伏区，理解灰线与大战残伤。
4. 最后回到核心维持区，面对法宝与地域未来。

## 4. 角色与功能地图的对应

- 守序飞升者：从现实承接区进入，对核心维持区形成迟到但决定性的理解。
- 剑器飞升者：更早与流动边界区和裂伤外缘发生高压接触。
- 愿力飞升者：能在现实承接区与核心维持区之间建立异质承认链观察。
- 中心执行者：天然靠近核心维持区与残秩序缓冲圈。
- 地方承接者：天然站在现实承接区。
- 流动节点者：天然站在流动边界区。
- 灰线历史位：天然站在失稳潜伏区与接引残忆触发点之间。

- 这张地图是功能地图，不是正式地理地图。
- 后续如需加具体地点，应先证明它归属于哪个功能区，避免地点乱长。
