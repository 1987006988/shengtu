# 05 历史索引

## 模块定位

`05_history` 是本项目的历史模块，用于维护宇宙历史层、九主世界格局历史层与故事主世界重点历史层的正式框架。

本模块服务整个宇宙设定仓库，不是单一主世界前传目录，也不应直接退化成游戏前史或事件流水账。

## 编写原则

- 先总史，再分史，再重点史。
- 先搭时代框架，不先写碎事件年表。
- 历史允许存在断层、争议、不同法统说法不一，不写成全知教科书。
- 当前阶段不展开 `3000` 个小世界的个体历史，只保留世界群与历史层级框架。

## 当前文件

- `01_eras_and_calendar.md`：历史时代划分与纪年口径。
- `02_cosmic_history.md`：宇宙总史，说明三层秩序、世界群格局、接引秩序与灰色小世界如何形成长期结构。
- `03_nine_main_worlds_history.md`：九主世界格局史，解释九主世界差异如何在历史中形成。
- `04_gulie_cantian_history.md`：古裂残天重点史，说明双极旧秩序、大战、残胜与修复期如何导向当下格局。
- `05_historical_faultlines.md`：历史断层、多版本记载与当前不应写死为唯一真相的部分。
- `06_nine_main_worlds_era_matrix.md`：九主世界在七大时代中的统一历史映射表，用于横向比较历史轨迹。
