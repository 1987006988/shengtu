# 01 时代划分与纪年

## 1. 文档用途

本文档用于给 `05_history` 提供统一的时代框架、先后关系与写作约束。

当前阶段优先确定“大时代”而非精确纪年；后续如有必要，可在不推翻时代框架的前提下补充纪年法与局部时间轴。

## 2. 纪年口径

- 当前阶段暂不拍死统一纪年法。
- 当前阶段优先使用“时代划分 + 相对先后关系”的历史组织方式。
- 若后续需要增加统一纪年，应服务于全宇宙历史书写，而不是只服务某一主世界或某一剧情阶段。

## 3. 历史时代划分

### 时代一：秩序奠基时代

- 一句话定义：三层宇宙的高低位秩序逐步稳定，世界层级与位格差异开始成为长期结构。
- 时代重点：小世界、主世界、更高层界域的层级关系不再只是散乱存在，而开始被更稳定地识别、区分与承受。
- 时代结束时：宇宙不再是单纯的混沌并置结构，而具备后续形成世界群、接引秩序与主世界格局的前提。

### 时代二：主世界成形时代

- 一句话定义：多个主世界逐步成形，并开始对各自可触达的小世界形成长期影响。
- 时代重点：主世界不再只是零散高位世界，而开始形成更稳定的世界群归属与统摄能力。
- 时代结束时：第二层世界的多元格局初具轮廓，主世界与小世界之间的位格关系进一步被结构化。

### 时代三：接引扩张时代

- 一句话定义：飞升、接引与上升通道从个别现象转为长期秩序结构。
- 时代重点：主世界对小世界的筛选、导入、归类与安置逐渐成体系，世界群关系更加稳定。
- 时代结束时：接引秩序成为高低位世界之间最重要的长期接口之一。

### 时代四：主世界分化时代

- 一句话定义：九主世界的统治形式、文明重心、仙凡关系与修行生态逐步拉开差异。
- 时代重点：同样处于主世界层，不同主世界沿着不同的秩序路径、社会组织方式与道路结构发展。
- 时代结束时：九主世界不再只是体量不同，而是成为历史路径明显有别的高位文明体。

### 时代五：裂变与争霸时代

- 一句话定义：主世界之间的隔离加深，竞争、断联、争夺与灰色世界增多。
- 时代重点：多主世界并存格局不再只是分化，而开始伴随更强的边界意识、世界群竞争与历史断层积累。
- 时代结束时：灰色小世界长期存在成为常态，多极隔离格局加速固化。

### 时代六：大战与重构时代

- 一句话定义：部分主世界经历高烈度冲突、旧秩序崩解与结构性重构。
- 时代重点：并非整个宇宙同时同形地进入同一事件，而是若干关键主世界在这一时期完成了决定后世格局的重大转折。
- 时代结束时：九主世界的强弱、新旧、完整度与统治样貌被进一步改写，古裂残天的旧日裂伤也在此阶段形成。

### 时代七：修复与并存时代（当前时代）

- 一句话定义：大战后的修复、再整合、隔离维持与多版本秩序并存共同构成当前宇宙。
- 时代重点：主世界总体仍在运转、研究、竞争与扩张，但许多旧裂伤、灰色区与历史断层仍未被真正消化。
- 时代结束时：当前仍在进行中，尚无统一终局口径。

## 4. 各时代的大致先后关系

- 秩序奠基时代 -> 主世界成形时代 -> 接引扩张时代 -> 主世界分化时代 -> 裂变与争霸时代 -> 大战与重构时代 -> 修复与并存时代（当前时代）
- 这是宇宙级共识框架，不等于每一个主世界都在同一节奏下同步经历所有变化。
- 后续具体历史书写，应优先说明“某一变化主要发生在哪一时代”，而不是急于填充精确年份。

## 5. 九主世界与古裂残天在时代框架中的位置

- 九主世界整体差异，主要在“主世界分化时代”后逐步定型。
- 灰色小世界长期存在，主要在“裂变与争霸时代”后成为稳定结构。
- 古裂残天的双极旧秩序形成与长期共治，可主要放在“主世界分化时代”与“裂变与争霸时代”之间书写。
- 古裂残天的高烈度大战与残胜，属于“大战与重构时代”的重点内容。
- 古裂残天当前的大势力主导、执行力衰减与四级社会模型，属于“修复与并存时代”的结构结果。

## 6. 本文件对后续历史写作的约束

- 不得跳过时代框架，直接把历史写成碎事件流水账。
- 不得把 `05_history` 直接写窄成古裂残天单世界前史。
- 不得一开始就给出过多精确年份与局部纪年，除非确有跨模块必要。
- 对同一历史变化，如存在断层、失落或法统分歧，应允许保留多种说法并在对应文档中标注。
- 后续细化历史时，应先补宇宙总史与九主世界格局史，再补单一主世界重点史。
