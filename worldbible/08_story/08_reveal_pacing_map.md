# 08 揭示节奏图

## 1. 文档用途

本文档用于整理哪些信息适合前段只隐约出现，哪些适合中段明确，哪些必须压到后段才完全打开。

## 2. 前段只应隐约出现的内容

- 接引法宝与主角团之间并非单纯被动接触，而是深层绑定。
- 此地并不是普通边荒，而是被大战推离常规秩序带的高位地域。
- 中心执行者背后并不是完整中心体系，而是困在此地的一支残余执行链。
- 裂史携行者知道的东西远多于其显性说出的部分。

## 3. 中段第一次明确的内容

- 这片地域内部仍存在真实社会、局部秩序与托底结构。
- 守护残余、反抗残余、小势力与地方共同体都在此地长期留有布局。
- 空间入口带、高风险秩序市场与流动通路是此地继续维持的重要装置。
- 主角团并不是在探索一片死地，而是在进入一个封闭而继续运转的局部世界。

## 4. 后段才完全打开的内容

- 这片地域原本是接引法宝布置之地。
- 大战如何把它连同法宝一起推入隔绝带。
- 主角团为何会在飞升途中与法宝绑定并获得复生能力。
- 地域是否重连、重组、继续封闭，为什么会变成主线最终命题。

## 5. 如何避免过早把底牌掀光

- 前两幕只让法宝显出“稳定 / 复生 / 残秩序核心”三层作用，不一次讲透全部来历。
- 第三幕才把法宝与大战、守护残余、反抗残余和主角团绑定的历史关系真正抬高。
- 第四幕不再用新真相取胜，而用代价比较与命题压缩收束。
