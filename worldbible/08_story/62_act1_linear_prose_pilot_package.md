# Act I 直线正文试点包

## 1. 文件定位

本文件不是主线正文开放令，也不是支线正文入口。  
它只服务两个严格受控目标：

1. 先把 `1–2` 条极小支线候选压成**范围定义**，而不是正文；
2. 再把 `CH01-S01–S02` 的一段**直线正文试点**压成可审样本。

本文件只用于证明：正文与极小扩容讨论也能继续服从  
`15 / 16 / 21 / 22 / 54 / 55 / 56 / 57 / 58`，  
而不是借试点把主线、支线或全面任务层一起放开。

## 5. 直线正文试点锁定

### 5.1 当前唯一正文试点范围

- 试点窗口：`CH01-S01–S02`
- 试点角色：`沈行策 / 桑沉岫`
- 试点目标：
  - 验证正文能否服从 `54 / 55` 的章场边界；
  - 验证正文能否只承接 `16 / 56` 允许的前段信息；
  - 验证正文能否在 `57 / 58` 允许的角色 / 状态增量内停住。

### 5.2 为什么只取 `CH01-S01–S02`

1. 这里只测试坠落、危险体感与第一次护撤，不碰第一次回返解释。
2. 这里只调用 `沈 / 桑` 两个前段接口，不提前把鹿门集、守衡司或白鹿半句拉进来。
3. 这里最容易看出正文是否会自发偷渡解释、抬高身份或越场推进。
