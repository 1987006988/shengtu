# 06 主线节拍骨架

## 1. 第一幕：坠落与封绝

### 幕名（暂定）

- 残秩序中的落地

### 开场状态

- 主角团在飞升途中与接引法宝绑定后坠入隔绝地域。
- 这里并未崩塌成纯废墟，而是残秩序、地方生存层与危险边界共同存在。

### 核心推进目标

- 建立主线舞台只是古裂残天内部的隔绝地域。
- 建立主角团为什么没有被外界大能直接碾碎。
- 建立法宝绑定、复生能力与地域稳定的基础关系。

### 玩家在这一幕主要接触到的世界层

- 生存承接带
- 残秩序带外缘
- 少量空间入口带表层

### 主导冲突类型

- 生存与适配冲突
- 阵营冲突
- 残秩序与现实承接冲突

### 第一重大认知翻转

- 这里不是普通废墟，也不是完整主世界中心区，而是一片高位残秩序仍在、却被切离常规秩序带的封闭地域。

### 幕结束时的结构变化

- 主角团完成最基础的落地与自保。
- 玩家理解这片地域的问题不是局部乱局，而是高位大战后的结构性封绝。

### 当前不应提前写死的部分

- 不提前写死第一幕的具体收束事件形式。

## 2. 第二幕：残存秩序与局部生存

### 幕名（暂定）

- 谁在托住这片地

### 开场状态

- 玩家已经理解这里是封闭高位地域，但尚未真正理解它靠什么活下来。

### 核心推进目标

- 打开本地聚落、小宗门、小势力、资源区、危险节点与入口带结构。
- 建立守护残余、地方承接层与流动网络如何共同托底。

### 玩家在这一幕主要接触到的世界层

- 生存承接带
- 流动通路带
- 空间入口带

### 主导冲突类型

- 地方生存冲突
- 流动网络冲突
- 残秩序与地方现实摩擦

### 第一重大认知翻转

- 这片地域能够苟续，不只是因为法宝没坏，也因为有人、有组织、有网络在持续承担成本。

### 幕结束时的结构变化

- 主角团开始拥有比第一幕更广的地域内部接口。
- 玩家意识到这片地并不是遗迹壳，而是一个带伤运转的局部世界。

### 当前不应提前写死的部分

- 不提前写死第二幕中哪条地方 / 边缘线索成为唯一主线支柱。

## 3. 第三幕：大战真相与法宝绑定

### 幕名（暂定）

- 裂伤开始说话

### 开场状态

- 玩家已见到残存社会与局部秩序，但对地域形成原因、法宝位置与复生根源仍不完整。

### 核心推进目标

- 把历史余波、反抗残余、守护残余与法宝绑定结构真正抬上桌。
- 让玩家意识到主角团已经被卷进这片地域的命运机制中。

### 玩家在这一幕主要接触到的世界层

- 空间入口带与附属空间接口
- 流动通路带深层
- 残秩序带回返
- 深层裂伤带边缘

### 主导冲突类型

- 历史冲突
- 灰线冲突
- 法宝归属与使用逻辑冲突

### 第一重大认知翻转

- 主角团并不是碰巧落到这里，而是在飞升途中与接引法宝绑定后，已经变成这片地域结构的一部分。

### 幕结束时的结构变化

- 玩家不再能把主线理解成逃出生天或局部求生。
- 主角团内部对法宝、守护、反抗与地域未来的分歧正式抬高。

### 当前不应提前写死的部分

- 不提前写死第三幕中哪一种历史解释占绝对主导。

## 4. 第四幕：地域命运的决断

### 幕名（暂定）

- 谁来承担重连的代价

### 开场状态

- 玩家已经理解这片地域为何存在、为何未崩、为何与主角团绑定。

### 核心推进目标

- 把“这片地域要不要重连、重启、继续封闭或重组”推到不可回避。
- 让此前的阵营冲突、历史冲突与角色分歧全部汇到“谁承担代价”上。

### 玩家在这一幕主要接触到的世界层

- 残秩序核心带
- 深层裂伤带与法宝核心接口
- 生存承接带回响层
- 少量外界威压回声

### 主导冲突类型

- 命题级冲突
- 秩序选择冲突
- 代价承担冲突

### 第一重大认知翻转

- 真正需要选择的不是“拯救主世界”，而是“这片地域未来要成为什么，以及谁来背负其重连或继续封闭的成本”。

### 幕结束时的结构变化

- 主角团对秩序、修复、封闭、重连与代价的理解完成从地域现实到命题决断的跃升。
- 玩家进入可承接具体剧情设计的结构终点。

### 当前不应提前写死的部分

- 不提前写死最终决断形式、最终对立排列与具体结局树。
