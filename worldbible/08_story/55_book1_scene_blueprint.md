# 第一部详细场景 / 事件蓝图文件

## 1. 文件定位

本文件把 18 章继续拆到关键场。  
场景不是对白样稿，而是后续任何正文 / 任务 / 关卡 / 演出继续施工时的硬骨架。

## 2. 场景写作总原则

- 每场必须有明确目标；
- 每场必须有明确冲突；
- 每场必须知道自己推进哪条线；
- 每场必须知道自己释放哪类信息；
- 每场结束必须自然推入下一场，不靠作者硬切。


## CH01《坠落不死》场景分解

### CH01-S01

- **地点**：黑砂外缘·裂风带
- **出场人物**：沈行策
- **场景目标**：建立坠落与异地失重感
- **场景冲突**：主角在飞升途中被扯离既定上行路径并坠入失稳外缘
- **推进线**：主线 / 世界线
- **释放信息**：主角不是主动来此；古裂残天的规则密度与小世界完全不同
- **情绪基调**：惊悸、迷失
- **场景结果**：主角重伤落地并立即发现此地危险会主动追人
- **如何进入下一场**：引出外缘猎手的第一视角

### CH01-S02

- **地点**：黑砂外缘·断脊沟
- **出场人物**：沈行策、桑沉岫、裂伤异兽
- **场景目标**：让白鹿原危险具体化
- **场景冲突**：桑沉岫必须决定救不救这个陌生人，同时异兽逼近
- **推进线**：主线 / 角色线
- **释放信息**：外缘并非无人地；桑沉岫代表最先被危险吞掉的人群
- **情绪基调**：紧迫、粗粝
- **场景结果**：桑沉岫带着主角奔逃，主角首次战斗失利
- **如何进入下一场**：导向第一次死亡

### CH01-S03

- **地点**：外缘残坛
- **出场人物**：沈行策、桑沉岫
- **场景目标**：引爆第一次回返
- **场景冲突**：主角死亡后在残坛边异常回返，桑沉岫必须决定是否上报
- **推进线**：主线 / 规则线
- **释放信息**：回返不是复活戏法，而像被某种地方结构强行拉回
- **情绪基调**：骇异、压迫
- **场景结果**：主角胸口出现回引印，桑沉岫确认事件已超出私人处理范围
- **如何进入下一场**：必须押送回鹿门集

### CH01-S04

- **地点**：通往鹿门集的护撤道
- **出场人物**：沈行策、桑沉岫、巡守队
- **场景目标**：完成第一章因果收束
- **场景冲突**：巡守队对主角身份分歧：杀、留、交给守衡司
- **推进线**：主线 / 世界线
- **释放信息**：白鹿原对异常个体的第一反应不是欢迎，而是风险管理
- **情绪基调**：戒备、僵硬
- **场景结果**：主角被定为‘待审风险人’，押送鹿门集
- **如何进入下一场**：进入地方现实层

- **推进线**：主线 / 世界线
- **释放信息**：共保会不是温情组织，而是不断做痛苦排序的现实机器
- **情绪基调**：压抑、现实
- **场景结果**：主角主动承担劳作换取临时停留

### CH02-S04

- **地点**：鹿门集议棚
- **出场人物**：沈行策、柳照禾、守衡司使者

- **推进线**：主线 / 阵营线
- **释放信息**：主角已不再只是个人，而是会影响结构的异常变量
- **情绪基调**：僵冷、被迫
- **场景结果**：柳照禾争取到由本地陪同押送的空间
- **如何进入下一场**：顾砺衡将在下一章接手

### CH03-S02

- **地点**：鹿门集北缓冲圈
- **出场人物**：沈行策、顾砺衡、白鹿之灵显化

- **场景冲突**：鹿门集与守衡司就谁陪同、谁承担路上损失争执不下
- **推进线**：主线 / 角色线
- **释放信息**：桑沉岫熟悉路险，柳照禾坚持有人替地方看着守衡司

- **情绪基调**：冷峻、悬压

- **如何进入下一场**：进入路权与流动层

## CH06《救一地还是稳一原》场景分解

### CH06-S01

- **地点**：风窖台核心裂口
- **出场人物**：沈行策、顾砺衡
- **场景目标**：先看结构优先线
- **场景冲突**：如果优先稳结构，可保更大片缓冲，但风窖台难民必须撤不出来
- **推进线**：主线 / 阵营线
- **释放信息**：守衡司选择不是无情，而是把更大的伤亡视作不可接受
- **情绪基调**：理性、残酷
- **场景结果**：主角真正理解顾砺衡为何总是先说整体
- **如何进入下一场**：转入人命线对照

### CH06-S02

- **地点**：风窖台疏散路
- **出场人物**：柳照禾、桑沉岫、难民
- **场景目标**：再看人命优先线
- **场景冲突**：如果优先抢人，可拖出更多活口，但整条缓冲带将长期恶化
- **推进线**：主线 / 角色线
- **释放信息**：人命优先也不是无代价善，它会把损失推到未来
- **情绪基调**：悲壮、匆忙
- **场景结果**：桑沉岫带人强行开路，柳照禾接住最坏后果
- **如何进入下一场**：回到主角总决断

### CH06-S03

- **地点**：白鹿显化的裂口边
- **出场人物**：沈行策、白鹿之灵、主角队
- **场景目标**：给出第一卷最大判断
- **场景冲突**：白鹿之灵只展示两条代价排序，不替主角做价值结论
- **推进线**：主线 / 规则线
- **释放信息**：白鹿并非神迹外挂，而是把‘哪一种死法更慢’摊开
- **情绪基调**：冷静、刺痛
- **场景结果**：主角做出保整体但尽量抢人、由自己承受回引反冲的方案
- **如何进入下一场**：代价兑现

### CH06-S04

- **地点**：风窖台废墟
- **出场人物**：主角队、幸存者
- **场景目标**：卷一收尾与下卷点火
- **场景冲突**：灾难过去，幸存者并不感激所有人；废墟中出现灰缄封记
- **推进线**：主线 / 阵营线
- **释放信息**：白鹿原的裂口有人在故意扩大
- **情绪基调**：苦涩、后震
- **场景结果**：众人确认白鹿原内部存在主动推灾之手
- **如何进入下一场**：第二卷从查内裂开始


## CH07《灰线不从外面来》场景分解

- **如何进入下一场**：进入第十五章


## CH15《你回来的代价》场景分解

- **场景目标**：让最重要的现实关系发声
- **场景冲突**：柳照禾反对把一个人写成新结构零件，桑沉岫则逼主角说清自己到底怕什么
- **推进线**：角色线 / 情感线

- **场景冲突**：三种未来方案第一次完整被摆出，但暂不做最终决定
