# 23 白鹿原关键节点

## 1. 文档用途

本文档用于给白鹿原建立一组可直接承接剧情推进的关键节点类型。

本文档强调“节点功能”，不强行发明正式专名。

## 2. 关键节点列表

### 节点一：法宝核心牵引面

- 所属结构：核心维持带
- 作用：决定白鹿原最深层结构是否仍被接引法宝托住。
- 故事用途：后段总冲突与最终选择必须回到这里。

### 节点二：残秩序缓冲圈

- 所属结构：核心维持带
- 作用：把高位压力削成白鹿原可承受的秩序外壳。
- 故事用途：中前段用于证明这里不是普通废墟。

### 节点三：主聚落承接面

- 所属结构：生存承接带
- 作用：聚集凡人、地方共同体、小势力与日常秩序。
- 故事用途：建立“这片地方有人活着，而且活得有成本”。

### 节点四：资源与风险交换带

- 所属结构：生存承接带
- 作用：把资源、消息、风险处理与地方权力绑在一起。
- 故事用途：展示为什么地方势力会依附坐大。

### 节点五：入口残带

- 所属结构：生存承接带与崩坏潜伏带交界
- 作用：维持有限流通，也是最敏感的边界带。
- 故事用途：流动节点者与灰线潜伏最容易在这里正面交锋。

### 节点六：裂伤外缘

- 所属结构：崩坏潜伏带
- 作用：把大战残伤、危险环境和高价值风险绑定在一起。
- 故事用途：承担探索、失稳、代价升级和真相触发。

### 节点七：隐匿潜伏层

- 所属结构：崩坏潜伏带
- 作用：容纳灰线活动、历史残忆、断裂承认链与反抗残余。
- 故事用途：中后段揭示灰线与大战真相的入口。

### 节点八：接引残忆触发点

- 所属结构：跨层节点
- 作用：连接白鹿之灵、法宝残忆和主角团绑定反应。
- 故事用途：推进第三幕“为什么是他们被卷进来”。

## 3. 节点之间的因果顺序

前段优先接触：

1. 主聚落承接面
2. 资源与风险交换带
3. 残秩序缓冲圈

中段逐步打开：

4. 入口残带
5. 裂伤外缘
6. 隐匿潜伏层

后段必须回收：

7. 接引残忆触发点
8. 法宝核心牵引面

## 4. 节点使用边界

- 不把节点写成地图打卡清单。
- 不把节点写成副本列表。
- 每个节点都必须至少同时承担“结构作用 + 冲突作用 + 揭示作用”中的两项。
