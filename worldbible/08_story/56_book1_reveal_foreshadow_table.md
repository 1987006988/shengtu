# 第一部详细揭示与伏笔表

## 1. 文件定位

本文件用于锁死信息顺序。  
它回答：

- 什么信息第几章可以出现；
- 什么信息必须后置；
- 哪些伏笔先埋，什么时候回收；
- 哪些真相只能由哪类角色 / 事件带出。

## 2. 使用原则

1. **先现实，后结构，再历史，最后代价比较。**
2. 任一揭示若不改变行动条件，应降级为环境文本或 Codex。
3. 任一伏笔若没有明确回收点，不得硬埋。
4. 任一真相若提前说满，必须视为蓝图事故。

## 4. 绝对禁止提前写明的内容

- 白鹿原被推离的完整古战全貌；
- 接引法宝的宇宙级来历；
- 主角回返的全部终局意义；
- 九主世界的全面政治布局；
- 白鹿之灵的更高层真实位阶。

## 5. 绝对不能缺失的提前铺垫

- 回引印与身体反冲；
- 白鹿显化只说半句；
- 路权与配额就是政治；
- 共保会不是纯善、守衡司不是纯恶；
- 灰缄线提出的问题不是假的；
- 历史材料也可能被安排留下。

## 6. 一句话口径

> 信息顺序就是《封绝之地》的第二剧情骨架；一旦信息顺序乱了，所有代价都会失真。
