# 22 隔绝地域结构性张力图

## 1. 文档用途

本文档用于整理隔绝地域内部最核心的长期结构性对抗关系。

本文档只写长期张力，不写具体阵营剧情、任务链、关卡流程或对白。

本文档内容属于第一部 / 当前故事强绑定设定层，不自动外推为整个 `古裂残天` 长期正典或宇宙长期正典。

当前第一部主舞台正式设定名为 `白鹿原`；“隔绝地域 / 封绝地域”是功能性描述，不替代正式地名。

接引法宝之灵正式显化为白鹿。白鹿之灵是主线核心装置的灵性接口，不是普通灵兽、独立神祇或 `古裂残天` 整体意志。

## 2. 当前最核心的六种长期张力

### 张力一：守护残余 vs 反抗残余

- 这是最直接、也最容易被误读成单纯阵营对打的长期张力。
- 其本质不是谁立刻彻底消灭谁，而是谁有资格定义这片地域未来该如何被继续维持、重连或改写。

主要落点：

- 残秩序缓冲区
- 隐匿区 / 灰线潜伏区
- 危险裂伤区边缘

功能权重提示：守护残余的结构控制权重最高，反抗残余的变量改写权重最高。

### 张力二：生存承接层 vs 法宝调用层

- 地方共同体、凡人聚落、小宗门与小势力需要稳定可活的现实空间。
- 但法宝越被深入调用，越可能撬动更高层结构，也越可能冲击当前现实平衡。

主要落点：

- 相对稳定生存区
- 残秩序缓冲区
- 法宝核心影响区

功能权重提示：地方承接层的现实托底权重最高，法宝调用层会直接冲击这种托底。

白鹿之灵位于法宝调用层与主角团复生机制之间。它可以识别接引对象、维系最低限度接引秩序并保存部分路径记忆，但不能替代地方承接层的现实托底。

### 张力三：封闭稳定 vs 重连冲动

- 当前封闭状态虽然压抑，但它也是现实适应后的脆弱秩序基础。
- 而任何重连冲动都可能意味着新的机会，也意味着重新暴露在主世界高位秩序与更大代价之下。

主要落点：

- 法宝核心影响区
- 残秩序缓冲区
- 通路 / 节点 / 入口残带

### 张力四：明面秩序 vs 灰线潜伏

- 明面秩序需要让地域至少维持“可活、可管、可承接”。
- 灰线结构则不断利用裂缝、错位、通路与承认盲区成长。

主要落点：

- 隐匿区 / 灰线潜伏区
- 通路 / 节点 / 入口残带
- 相对稳定生存区边缘

### 张力五：中心残秩序惯性 vs 地方自组织惯性

- 地域内部仍有中心残秩序的投影、旧制承认链与守护残余逻辑。
- 但现实活下来的更多依靠地方承接层、地方共同体、小宗门与小势力。

主要落点：

- 残秩序缓冲区
- 相对稳定生存区
- 通路 / 节点 / 入口残带

### 张力六：流通续命 vs 裂伤封锁

- 地域要继续存在，就必须保留最低限度的流通与节点联络。
- 但裂伤区、失稳区与结构封锁又不断切断、扭曲甚至吞没这种流通。

主要落点：

- 通路 / 节点 / 入口残带
- 危险裂伤区
- 相对稳定生存区边缘

## 3. 哪些张力最适合作为前中后段推进动力

三类力量功能权重已正式收口为：

- 地方承接层：现实托底权重最高，决定今天还能不能活。
- 守护残余：结构控制权重最高，决定整体还维不维持得住。
- 反抗残余：变量改写权重最高，决定未来会不会被改写。

这轮后视为稳定执行口径，不写数字比例，不再重复摇摆。

### 前段更适合上桌的张力

- 生存承接层 vs 法宝调用层
- 流通续命 vs 裂伤封锁
- 明面秩序 vs 灰线潜伏

原因是：

- 它们更容易先以现实压力、生存摩擦与局部异常形式显形。
- 不需要过早抬出整个地域最终命运，就能让主角团立刻感受到这片地方为什么危险又还能活。

### 中段更适合抬高的张力

- 守护残余 vs 反抗残余
- 中心残秩序惯性 vs 地方自组织惯性
- 明面秩序 vs 灰线潜伏

原因是：

- 到中段，主角团应已不只看到生存表层，而开始看到“谁在托底、谁在渗透、谁在借结构生长”。

### 后段更适合承接的张力

- 封闭稳定 vs 重连冲动
- 生存承接层 vs 法宝调用层

原因是：

- 这两组张力最接近当前故事主轴“地域未来如何被决定，以及由谁承担代价”。

## 4. 哪些张力暂时只能做背景压力

当前不宜过早推到台前正面展开的，主要包括：

- 封闭稳定 vs 重连冲动的终局版本
- 守护残余 vs 反抗残余的最终形态
- 中心残秩序惯性 vs 地方自组织惯性的最终胜负

这些张力可以提前投下压力，但不应在前段就被简化成明确阵营站队题，否则会过早耗掉隔绝地域的结构厚度。

## 5. 当前不应提前写死的部分

- 不提前写死六种张力在四幕中的精确占比。
- 不提前写死哪一种张力最终压过全部其他张力。
- 不提前写死哪些张力会直接转入任务层，哪些只保留为背景压力。
- 不提前写死守护残余、反抗残余、地方承接层与灰线结构的最终排序。
- 不把白鹿之灵写成万能真相解释器、冲突解决者或古裂残天整体意志。
