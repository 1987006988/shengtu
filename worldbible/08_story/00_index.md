# 00 故事模块索引

## 1. 模块定位

`08_story` 当前默认是《升途：封绝之地》的蓝图层与受控任务层入口。

当前真实阶段不是旧的 game narrative 生产期，也不是旧 batch / review / final acceptance 口径下的任务文本工程阶段，而是：

> `Act I 任务层开放阶段`

因此，本模块当前默认处理的是：

- 第一部总纲与第一卷总纲
- 卷 / 幕 / 章 / 场蓝图
- 揭示与伏笔表
- 角色推进表
- 世界状态变化表
- `A1 / CH01–CH03` 已通过审计的主线正文与主线任务层
- `A2 / CH04–CH06` 已通过审计的主线正文与主线任务层
- `SQ-SCOPE-01` 已通过审计的单条支线试点
- `SQ-SCOPE-02` 已通过审计的单条支线试点

当前仍不得把本模块写成“全面任务层已开放”或“全书正文已开放”。  
接下来的默认工作不是回头重开旧 batch，而是进入“全面开放前总验证链”的受控校验。
