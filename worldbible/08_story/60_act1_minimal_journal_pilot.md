# Act I 最小 Journal 试点文件

## 1. 文件定位

本文件只服务 `CH01–CH02` 的最小 `Journal` 样例。  
它不是全面日志系统，也不是整部任务日志总包。

## 2. 使用边界

- 只记录玩家当前已经知道、且会改变行动判断的信息
- 只服务 `MQ-ACT1-01A / 01B / 02A / 02B`
- 不得提前写入白鹿原成因、白鹿之灵解释权、古战前史或后段答案

## 3. Active 条目

### 3.1 `JNL-ACT1-01《坠落不死》`

**当前目标**

- 跟着外缘猎手活下来
- 搞清楚为什么自己死后会被拉回来
- 在鹿门集证明自己不是会立刻害死所有人的麻烦

**当前认知**

- 白鹿原不是死地，这里还有很多人靠配额和护撤活着
- 自己的异常回返已经被人看见
- 被留下，不代表被接纳

### 3.2 `JNL-ACT1-02《被押送的人》`

**当前目标**

- 在押送途中别再制造新的风险
- 进入鹿门集并接受最基本的风审

**当前认知**

- 回引印已经让巡守队确认自己属于异常对象
- 白鹿原对异常者先做管理，不先做安慰

### 3.3 `JNL-ACT1-03《鹿门的价钱》`

**当前目标**

- 在照护棚与账本室里换到一段临时停留
- 看懂谁在为谁的活路付账

**当前认知**

- 鹿门集的承接位、基础补给额度、工分和承接顺位都很紧
- 自己被留下，会让别人的日子更紧一点

### 3.4 `JNL-ACT1-04《移交前夜》`

**当前目标**

- 在议棚冲突里保住最小本地看护权
- 进入守衡司结构审查前状态

**当前认知**

- 本地并不想失去解释权
- 自己已经被视为必须移交的结构异常变量
