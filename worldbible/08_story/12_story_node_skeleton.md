# 12 剧情节点骨架

## 1. 文档用途

本文档用于把四幕分段骨架进一步拆成可持续扩写、但尚未任务化的关键剧情节点。

当前阶段只回答每个节点负责推进什么、抬高什么、揭示什么、把玩家推向哪里，不写任务清单、对白草稿、详细关卡流程或分支脚本。

## 第一幕：坠落与封绝

- 所属剧情段：坠落适配段

- 本节点新增的世界认识：这里仍是高位环境，但已经脱离常规秩序带。
- 本节点如何把玩家推向下一节点 / 下一段：先确立“这里没塌”，再追问“为什么没塌”。

### 代价聚焦段

#### 27. 法宝回收节点
- 所属幕：第四幕
- 所属剧情段：代价聚焦段
- 本节点核心问题：为什么所有问题最终都回到法宝与残秩序核心。
- 本节点主导冲突层：命题级冲突的现实落点
- 本节点主要抬高的角色：中心执行者、守序飞升者
- 本节点新增的世界认识：要决定此地未来，必须先直面法宝核心而非绕开它。
- 本节点如何把玩家推向下一节点 / 下一段：推进到代价具体化。
- 当前不应提前写死的部分：不写死回收形式。

#### 28. 承压面聚拢节点
- 所属幕：第四幕
- 所属剧情段：代价聚焦段
- 本节点核心问题：谁在为此地继续存在承担现实压力。
- 本节点主导冲突层：结构代价冲突
- 本节点主要抬高的角色：地方承接者、流动节点者、中心执行者
- 本节点新增的世界认识：代价最后会落在本地社会、入口带和残秩序执行链上。
- 本节点如何把玩家推向下一节点 / 下一段：把外部问题压成内部立场问题。
- 当前不应提前写死的部分：不写死最重承压者。

### 立场压缩段

#### 29. 代价镜像节点
- 所属幕：第四幕
- 所属剧情段：立场压缩段
- 本节点核心问题：为什么重连、继续封闭、重启、重组都各有代价承担者。
- 本节点主导冲突层：命题冲突 / 镜像冲突
- 本节点主要抬高的角色：裂史携行者、愿力飞升者
- 本节点新增的世界认识：不存在零代价方案，只存在不同承担分配。
- 本节点如何把玩家推向下一节点 / 下一段：推进到团队分歧实化。
- 当前不应提前写死的部分：不写死所有镜像路径细节。

#### 30. 团队分歧实化节点
- 所属幕：第四幕
- 所属剧情段：立场压缩段
- 本节点核心问题：当代价摆到明面上时，主角团为何不能自然统一。
- 本节点主导冲突层：角色内部冲突与命题冲突叠合
- 本节点主要抬高的角色：七位角色全部抬高，中心执行者、裂史携行者、守序飞升者最显著
- 本节点新增的世界认识：同样看见全部现实，不意味着会给出同样答案。
- 本节点如何把玩家推向下一节点 / 下一段：把故事压到路径比较。
- 当前不应提前写死的部分：不写死分歧排列。

### 重连 / 封闭比较段

#### 31. 重连代价节点
- 所属幕：第四幕
- 所属剧情段：重连 / 封闭比较段
- 本节点核心问题：让此地重回主世界秩序意味着什么。
- 本节点主导冲突层：秩序选择冲突
- 本节点主要抬高的角色：中心执行者、守序飞升者
- 本节点新增的世界认识：重连不是救援按钮，而是会重排承认链、权力链与承压面。
- 本节点如何把玩家推向下一节点 / 下一段：推进到继续封闭或重组的比较。
- 当前不应提前写死的部分：不写死重连方案细节。

#### 32. 封闭 / 重组代价节点
- 所属幕：第四幕
- 所属剧情段：重连 / 封闭比较段
- 本节点核心问题：继续封闭或重组此地又意味着什么。
- 本节点主导冲突层：秩序选择冲突
- 本节点主要抬高的角色：愿力飞升者、裂史携行者、地方承接者
- 本节点新增的世界认识：不重连并不等于没有代价，重组也不等于天然更正义。
- 本节点如何把玩家推向下一节点 / 下一段：把路径比较压到最终命题。
- 当前不应提前写死的部分：不写死候选路径具体数量。

- 所属剧情段：命题决断段

- 本节点主导冲突层：命题冲突终层
