# 29B 第一款游戏场景卡包：Act 3 / Act 4 / Epilogue

## 1. 文档定位

本文档对应 `B04：重建游戏级场景卡 B 与总索引` 的主输出之一。

它负责把 `27_volume1_plot_spine.md` 与 `28_volume1_chapter_outline.md` 中属于 Act 3、Act 4 与 Epilogue 的部分，展开成可供任务架构、正文扩写、Journal 更新、同伴插话与状态机调用的场景卡。

本文档不是小说章节复述，不是对白正文，也不是任务 ID 表。它必须继续服务 `白鹿原` 第一款游戏尺度，不上浮为整个 `古裂残天` 的总命运线。

## 2. 本批范围与边界

- 只覆盖 `MQ25–MQ48`，即第 `08–17` 章。
- 直接继承 `29a_volume1_scene_cards_act1_act2.md` 的中段状态积累，不回头重做第二幕铺垫。
- 不进入 B05 的主线任务 ID、主线正文、同伴文本、支线、Codex、Journal、Item 或 System 文本。
- 白鹿之灵继续只承担识别、示警、残片与边界反馈，不得越权为路线裁判。

## 3. 场景卡使用规则

- 每张卡都必须说明：存在理由、核心冲突、前置状态、成功结果、失败 / 变体结果、状态输出、后续挂钩。
- Act 3 的重点是：灰线显形、异质承认、法宝翻面、绑定回返、认知崩解。
- Act 4 / Epilogue 的重点是：法宝回收、承压面聚拢、立场压缩、路径比较、受限未来。
