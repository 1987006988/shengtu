# Act I 最小 Codex 试点文件

## 1. 文件定位

本文件只服务 `CH01–CH02` 的最小 `Codex` 样例。  
它只提供前段行动导向信息，不承担总解释功能。

## 2. 总规则

1. 条目只允许写前段可知、样本级、现象级信息。
2. 条目应帮助玩家理解“此刻该怎样看待风险与处境”，而不是解释宇宙真相。
3. 不得提前释放白鹿原成因、白鹿之灵完整解释权、古战前史、远端主世界全貌或终局答案。

## 3. 最小条目

### 3.1 `CDX-ACT1-001《白鹿原》`

**可见定义**

一片还在勉强活着的隔绝地带。  
这里还有人，但每一份护撤、病床、配额和安置位都很紧。

**当前允许知道**

- 这里不是死地，也不是安稳地
- 很多人靠持续分配、延后和看护活到今天
- 每多留下一个人，别人的日子就会更紧一点

**当前明确不知道**

- 白鹿原为什么会变成这样
- 它与更大主世界的完整关系
- 它与白鹿之灵的完整关系

### 3.2 `CDX-ACT1-002《黑砂外缘》`

白鹿原最容易先把人逼出原形的外缘带。  
这里先考验的不是理想，而是你能不能在下一次裂风来前活着挪到能站住的地方。

### 3.3 `CDX-ACT1-003《鹿门集》`

白鹿原的主聚落与托底面。  
病床、配给、工分、安置与移交流程，决定了这里“今天还能活着”到底要付什么价。

### 3.4 `CDX-ACT1-004《回引印》`

主角胸口出现的鹿蹄状异常印记。  
当前只知道它与主角的异常回返有关，也让白鹿原的巡守者把主角视为结构相关者。

3. 不得让条目替代 `15 / 16 / 21 / 22 / 54 / 55 / 56 / 57 / 58` 的上游定义。
