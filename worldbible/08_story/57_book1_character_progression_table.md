# 第一部详细角色推进表

## 1. 文件定位

本文件是第一部章场级角色推进控制表。  
它的作用是防止：

- 某个角色前段抢太多；
- 某个角色该抬权时没有抬；
- 某个角色过早翻面或过早说满；
- 多个角色抢同一功能；
- 角色推进表反向改写 `54 / 55 / 56` 已锁死的章场承压链、揭示顺序与越权边界。

   - 本章 / 本场到底抬高了哪一席位。
