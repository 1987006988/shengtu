# 第一部详细角色推进表

## 1. 文件定位

本文件是第一部章节级角色控制表。  
它的作用是防止：

- 某个角色前段抢太多；
- 某个角色该抬权时没有抬；
- 某个角色过早翻面或过早说满；
- 多个角色抢同一功能。

## 3. 阶段性抬权规则

### 前段（CH01–CH06）
主抬：
- 沈行策
- 柳照禾
- 顾砺衡
- 桑沉岫

次抬：
- 祁渡川

压后：
- 宁折羽
- 穆问碑

### 中段（CH07–CH12）
主抬：
- 宁折羽
- 穆问碑
- 祁渡川
- 柳照禾

持续抬：
- 沈行策
- 顾砺衡

### 后段（CH13–CH18）
主抬：
- 沈行策
- 顾砺衡
- 柳照禾
- 七席位群像比较

功能性抬：
- 祁渡川（路权与外联）
- 宁折羽（反向未来压力）
- 穆问碑（合法性与证据）
- 桑沉岫（失地与身体代价）

## 4. 角色推进硬约束

- 沈行策：不能用升级感替代代价感。
- 柳照禾：不能只做善良发言人，必须承担配给与排序的冷面。
- 顾砺衡：不能太早洗白，也不能太晚松动。
- 祁渡川：不能退化成气氛组。
- 宁折羽：不能一章洗白，一章讲完所有灰线真相。
- 穆问碑：不能把中前段写成历史 lecture。
- 桑沉岫：不能只在打架场有用。

## 5. 一句话口径

> 第一部角色推进的关键，不是人人都很丰满，而是人人在正确时机承担了不可替代的功能。
