# 21 隔绝地域结构总图

## 1. 文档用途

本文档用于把《升途：封绝之地》当前主舞台 `白鹿原`，也就是 `古裂残天` 内部这片隔绝地域，从“概念成立”推进到“结构可用”的第一轮框架。

本文档内容属于第一部 / 当前故事强绑定设定层，不自动外推为整个 `古裂残天` 长期正典或宇宙长期正典。

本文档只处理结构层，不写具体任务、对白、详细地图点位、关卡流程或事件链。

## 2. 这片隔绝地域在古裂残天中的位置性质

这片地域不是普通边地、普通遗迹或主世界内部常见封闭区，而是：

- 正式设定名为 `白鹿原`。
- “隔绝地域 / 封绝地域”是功能性描述，不替代正式地名；`离序残域` 不作为正式地名，仅可作为工作代号 / 档案代号 / 描述性标签保留。
- `古裂残天` 在高烈度大战后，被推离主世界常规秩序带的一片高位残存地域。
- 它仍属于 `古裂残天` 内部结构的一部分，但已不再处于主世界常态运行带中。
- 它与外界并非完全断绝，而是处于“难以正常接回、也难以被彻底放弃”的悬置状态。
- 它既保留了高位主世界残秩序的厚度，也承受着长期脱离常规秩序带的裂伤压力。

因此，这片地域更适合被理解为：

- 一个被大战推离后的高位残存结构；
- 一个仍被接引法宝与残秩序勉强托住的封闭地域；
- 一个对主角团仍然可介入、但对整个主世界来说成本极高、难以直接重收的局部命运带。

## 3. 地域当前整体状态

### 3.1 为什么没有彻底崩塌

这片地域没有彻底崩塌，至少依赖以下几层共同托底：

- 接引法宝仍在发挥核心结构支点作用。
- 大战遗留的残秩序并未完全散尽，仍形成若干可维持承认链与基本规则稳定的区段。
- 地域内部仍保留了一批能够承接现实生存的聚落、共同体、小宗门、小势力与地方性秩序。
- 某些通路、节点、入口残带仍能维持有限流通与资源循环，避免整片地域彻底失血。
- 守护残余与反抗残余虽然相互对抗，但都没有能力或意愿把整片地域直接拖入完全毁灭。

### 3.2 为什么无法正常回归主世界秩序带

它又不能正常回归主世界常规秩序带，主要因为：

- 大战留下的是结构性推离，而不是单一裂口。
- 外界常规秩序带与此地之间的接回条件已被严重破坏。
- 地域内部许多结构已围绕“残存托底”重新形成脆弱平衡，强行接回可能先导致局部失稳。
- 接引法宝与此地的关系已不只是“通道钥匙”，而是直接嵌入地域存续本身。
- 守护残余、反抗残余、地方共同体与灰线网络都已围绕当前封闭状态形成既得结构，不存在无代价回归。

## 4. 地域内部的核心结构层

当前正式采用“三带总括 + 六层执行模型”理解 `白鹿原`。

三带总括用于快速判断大结构功能；六层结构保留为详细执行模型。

### 4.1 核心维持带

核心维持带负责回答“整体还维不维持得住”。

它包括：

- 法宝核心影响区
- 残秩序缓冲区

### 4.2 生存承接带

生存承接带负责回答“今天还能不能活”。

它包括：

- 相对稳定生存区
- 通路 / 节点 / 入口残带

### 4.3 崩坏潜伏带

崩坏潜伏带负责回答“未来会不会被撕开、改写或拖向失稳”。

它包括：

- 危险裂伤区
- 隐匿区 / 灰线潜伏区

## 5. 六层执行模型

### 5.1 法宝核心影响区

- 最靠近接引法宝核心作用链条的结构层。
- 这里不是单纯“法宝所在地”，而是法宝承认、复生根源、结构托底与深层秩序牵引最强的区域带。
- 其最大特点是高位性仍强、代价也最高，不适合大规模常态生存，但决定整片地域的最终走向。
- 布置在 `白鹿原`、承担接引功能的法宝之灵，其显化形态正式拍板为 `白鹿`。

### 5.1.1 接引法宝之灵的白鹿显化

白鹿是接引法宝之灵的显化形态，是主线核心装置的灵性接口。

它不是普通灵兽，不是独立神祇，也不是 `古裂残天` 整体意志。

它能做的事：

- 识别飞升者与接引对象。
- 维系 `白鹿原` 与接引法宝相关的最低限度秩序。
- 作为主角团复生机制的直接接口。
- 保留部分大战前后的接引体系残忆、路径记忆与秩序碎片。

它不能做的事：

- 不能直接替主角团解决结构性冲突。
- 不能无限制复生而没有代价。
- 不能取代守护残余、反抗残余和地方承接层的现实作用。
- 不能作为万能真相解释器。
- 不能代表整个 `古裂残天` 意志。

### 5.2 残秩序缓冲区

- 介于法宝核心影响区与现实生存区之间的过渡层。
- 这里仍受旧秩序与法宝影响，但已不是纯高位核心层。
- 它的主要作用是把深层残秩序的压力缓冲下来，避免整片地域直接坠入全面失稳。

### 5.3 相对稳定生存区

- 当前最适合凡人聚落、小宗门、小势力与地方共同体长期承接现实生存的结构层。
- 这里不是绝对安全区，而是“仍有风险、但可以形成日常秩序”的生活带。
- 它们依赖法宝托底、残秩序余波、地方组织能力与有限流通共同维持。

### 5.4 通路 / 节点 / 入口残带

- 指仍能维持有限流通、资源交换、风险转移与结构联络的残存节点网络。
- 这些残带不等于完整交通体系，而是断续、稀缺、代价高的存续脉络。
- 它们决定了这片地域不会彻底碎成互不往来的死块。

### 5.5 危险裂伤区

- 大战后遗留下来的高风险结构层。
- 这里最接近失稳、错位、坠落、规则紊乱与空间撕裂带。
- 它既是生存威胁来源，也是资源、遗产、秘密与灰线通路的高密度风险区。

### 5.6 隐匿区 / 灰线潜伏区

- 依附于裂伤、错位、入口残带、旧制残片或局部承认盲区而长期存在的低可见层。
- 它不必是单独地理块，更像分布在若干层之间的潜伏带。
- 这里最适合藏匿灰线活动、反抗残余布局、旧敌残片和难以公开显形的结构性存在。

## 6. 各结构层的功能差异

- 法宝核心影响区：决定“这片地域还能否继续存在”。
- 残秩序缓冲区：决定“高位残秩序如何不直接压垮下层现实”。
- 相对稳定生存区：决定“人怎么活下去，地方秩序如何成立”。
- 通路 / 节点 / 入口残带：决定“东西怎么流，消息怎么传，资源怎么续”。
- 危险裂伤区：决定“这片地域为什么永远处于高压之下，也为什么仍不断生出争夺与秘密”。
- 隐匿区 / 灰线潜伏区：决定“哪些力量能够长期潜伏、绕开明面秩序并等待时机”。

## 7. 哪些结构层最适合承载哪些存在

### 凡人聚落

- 主要落在相对稳定生存区。
- 次级依附于残秩序缓冲区边缘与部分通路节点带。

### 小宗门 / 小势力

- 主要落在相对稳定生存区与通路 / 节点 / 入口残带交界处。
- 少数会借残秩序缓冲区边缘获取更高位的承认残片或旧制资源。

### 守护残余

- 更适合依附法宝核心影响区与残秩序缓冲区。
- 其明面活动更容易借助残秩序与旧制承认链维持合法性。
- 在功能权重上，守护残余的结构控制权重最高，决定整体还维不维持得住。

### 反抗残余

- 更适合依附隐匿区 / 灰线潜伏区、危险裂伤区边缘与部分通路残带。
- 它们依赖裂缝、错位与明面秩序难以完全覆盖的结构空间生存。
- 在功能权重上，反抗残余的变量改写权重最高，决定未来会不会被重新改写。

### 资源争夺

- 最容易集中在危险裂伤区边缘、通路 / 节点 / 入口残带，以及残秩序缓冲区的高价值边界层。

### 灰线活动

- 最适合落在隐匿区 / 灰线潜伏区。
- 但其生长通常依赖通路 / 节点 / 入口残带与相对稳定生存区的现实承接面。

## 8. 结构层之间的基本关系

- 法宝核心影响区提供最深层托底，但不能直接替代现实社会。
- 残秩序缓冲区把高位压力消化成可被下层承接的形式。
- 相对稳定生存区负责把“还能活”落实成“能长期活”。
- 通路 / 节点 / 入口残带负责让各层不至于彻底断裂。
- 危险裂伤区持续制造威胁、诱因和结构性不稳定。
- 隐匿区 / 灰线潜伏区则使这片地域永远不可能只有单线明面秩序。

功能权重口径正式收口为：

- 地方承接层：现实托底权重最高，决定今天还能不能活。
- 守护残余：结构控制权重最高，决定整体还维不维持得住。
- 反抗残余：变量改写权重最高，决定未来会不会被重新改写。
