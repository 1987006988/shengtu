# Act I 主线正文生产包

## 1. 文件定位

本文件只服务 `Act I / A1 / CH01–CH03` 的主线正文生产修订。  
它不是全书开放令，也不是新增审查链。

本文件只证明一件事：

> `CH01–CH03` 的主线正文，是否能在不越过 `15 / 16 / 21 / 22 / 54 / 55 / 56 / 57 / 58` 的前提下稳定落地。

- 只写 `CH01《坠落不死》`

- 只写 `CH03《白鹿只说半句》`

## 5. 正文硬规则

1. 正文只能按 `55` 已落库的场景顺序推进，不得换序。
2. 正文只能释放 `56` 允许的前段信息，不得补发高位解释。
3. 正文只能抬 `沈 / 柳 / 顾 / 桑` 的 `A1` 窗口权重，不得提前抬 `祁 / 宁 / 穆`。
4. 正文只能锁定 `58` 在 `CH01–CH03` 允许出现的状态增量，不得顺手推进 `CH04` 的道路政治。
5. 正文语言可以更细，但结论层不得比蓝图更早。

## 6. 正文禁止事项

1. 不得把主角写成天命开局或默认救世主。
2. 不得把白鹿之灵写成全知说明器。
3. 不得把守衡司提前写成绝对反派。
4. 不得把回返写成无代价复活戏法。
5. 不得把 `A1` 收束偷换成 `Act I` 全段主线正文已开放。

## 7. 审收目标

本包的通过标准不是“正文好不好看”，而是：

1. `CH01–CH03` 是否稳定停在 `A1`；
2. 三章正文是否仍服从 `54 / 55 / 56 / 57 / 58`；
3. 是否已足以作为 `73_act1_mainline_task_package.md` 的前置门槛。

## 8. 一句话口径

> 这组文件不是开全书正文，而是只给 `A1 / CH01–CH03` 做一轮现役主线正文修订闭环。
