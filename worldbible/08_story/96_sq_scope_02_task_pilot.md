本文件只服务 `SQ-SCOPE-02《断路前的空名册》`。  
它只允许停在 `CH04-S01–S02`，只服务 `A2` 窗口。

> `95_a2_mainline_task_review.md = PASS`

没有这个 `PASS`，本试点不应被解冻。

## 2. 任务定义

| 字段 | 内容 |
| --- | --- |
| `sq_id` | `SQ-A2-01《断路前的空名册》` |
| `chapter_window` | `CH04-S01–CH04-S02` |
| `attach_chain` | `断引栈道前哨 -> 旧接引节点残桥` |
| `lead_roles` | `沈行策 / 顾砺衡 / 祁渡川` |
| `entry_condition` | 完成 `MQ-A2-04A` 的前哨谈判起段，且队伍已被允许带着修补件进入残桥窗口 |
| `core_function` | 放大“谁有资格占用这一次护送空位、谁会在名册里被写上或被抹掉” |
| `objective_chain` | 协助祁渡川核对前哨路号 -> 发现一格被刻意留空的护送位 -> 在旧接引节点补缝前替这格空位找到临时承担人，并理解“空着不是失误，而是方便事后改写责任” |
| `completion_condition` | 主角理解“今天能被写进名册，本身就是被别人让出来的一次窗口”，并接受自己不是天然该占这格位置的人 |
| `allowed_reveals` | 路权会重排活路、护送空位稀缺、空名册可以被拿来抹平责任、祁渡川只负责把现实代价讲清 |
| `forbidden_reveals` | 灰缄线全貌、自由开路口径、完整外部世界、碑林历史线、主角无代价开路 |
